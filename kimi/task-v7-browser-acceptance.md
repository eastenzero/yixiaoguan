# 任务: V7 教师端 Browser Agent 验收

## 目标
使用 Agent Browser 技能，逐页访问教师端 H5 应用，截图并验证每个页面是否正常渲染。

## 应用地址
http://localhost:5175

## 验收路径

### 第1步: 登录页
1. 打开 `http://localhost:5175/#/pages/login/index`
2. 截图
3. 检查:
   - 是否有渐变背景
   - 是否有登录卡片（医小管 教师工作台 标题）
   - 是否有用户名输入框、密码输入框
   - 是否有验证码输入框和验证码图片
   - 是否有登录按钮
   - 是否有其他登录方式（QR码、指纹图标）

### 第2步: 工作台页
1. 打开 `http://localhost:5175/#/pages/dashboard/index`
2. 截图
3. 检查:
   - 是否有自定义顶栏（工作台 + 通知铃铛）
   - 是否有欢迎横幅（渐变卡片）
   - 是否有快捷操作按钮（新建知识、发布通知、数据报告、系统设置）
   - 是否有统计网格
   - 是否有待处理提问列表
   - 是否有底部导航栏（4个tab）

### 第3步: 学生提问列表
1. 打开 `http://localhost:5175/#/pages/questions/index`
2. 截图
3. 检查:
   - 是否有顶部导航栏（学生提问）
   - 是否有筛选标签（全部/待处理/处理中/已解决）
   - 是否有提问卡片列表（或空状态提示）
   - 是否有底部导航栏

### 第4步: 提问详情
1. 打开 `http://localhost:5175/#/pages/questions/detail`
2. 截图
3. 检查:
   - 是否有顶部导航栏（提问详情）
   - 是否有学生信息区域
   - 是否有对话/问题内容区域
   - 是否有底部操作按钮

### 第5步: 知识库列表
1. 打开 `http://localhost:5175/#/pages/knowledge/index`
2. 截图
3. 检查:
   - 是否有搜索栏
   - 是否有分类标签（全部/教务管理/学生服务/生活指南）
   - 是否有知识卡片列表（或空状态提示）
   - 是否有底部导航栏

### 第6步: 知识详情
1. 打开 `http://localhost:5175/#/pages/knowledge/detail`
2. 截图
3. 检查:
   - 是否有标题区域
   - 是否有文章正文区
   - 是否有底部操作栏（下线/编辑按钮）

### 第7步: 个人中心
1. 打开 `http://localhost:5175/#/pages/profile/index`
2. 截图
3. 检查:
   - 是否有个人信息渐变卡片（头像、姓名、职称）
   - 是否有统计网格（3列）
   - 是否有系统设置列表（通知提醒/声音提示/AI自动回复 开关）
   - 是否有退出登录按钮
   - 是否有底部导航栏

## 输出要求

对每个页面输出以下格式:

```
### 页面名称
- URL: xxx
- 渲染状态: ✅正常 / ❌白屏 / ⚠️部分渲染
- 截图: [已截取]
- UI 元素检查:
  - [元素1]: ✅ / ❌
  - [元素2]: ✅ / ❌
  - ...
- 问题: 无 / [具体问题描述]
```

最后输出汇总表格:

| 页面 | 状态 | 问题数 |
|------|------|--------|
| 登录页 | ✅/❌ | N |
| ... | ... | ... |
