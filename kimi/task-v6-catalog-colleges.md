# T3 任务: 教学机构子站点索引

## 任务目标
访问 https://www.sdfmu.edu.cn/jgsz1/jxjg.htm ，提取该页面上所有学院/教学机构的名称和链接，生成一份完整索引表。

## 重要说明
- **本任务只做索引**，不需要深入每个学院网站
- 只需收集：学院名称、官网 URL、是否可访问（HTTP 状态码）
- 如果页面有分类（如医学类、理工类等），保留分类信息

## 执行步骤

1. 用 Playwright 访问 https://www.sdfmu.edu.cn/jgsz1/jxjg.htm
2. 提取页面上所有学院/教学机构的超链接
3. 对每个链接做一次 HEAD 请求，确认是否可访问
4. 生成索引表

## 输出格式

写入 `kimi/report-9-college-index.md`：

```markdown
# 教学机构子站点索引

**来源页面**: https://www.sdfmu.edu.cn/jgsz1/jxjg.htm

## 学院列表

|------|---------|-----|--------|------|
| 1 | XXX学院 | https://xxx.sdfmu.edu.cn | ✅/❌ | ... |
| ... | ... | ... | ... | ... |

## 统计
- 总数: N 个
- 可访问: N 个
- 不可访问: N 个

## 分类汇总（如页面有分类）
```

请开始执行。
