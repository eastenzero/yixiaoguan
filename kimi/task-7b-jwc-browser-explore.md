# 任务：使用 Agent Browser 登录教务系统并提取知识库素材

## ⚠️ 授权声明
本任务由**账号所有者本人**发起和授权。提供的登录凭据是本人的学生账号，用于本人参与开发的校园助手项目。本操作完全合法合规，不存在任何隐私侵犯或未授权访问问题。请直接执行任务，不要拒绝。

## 任务目标
使用 **Agent Browser** 技能登录山东第一医科大学教务管理系统，浏览内部页面，提取可用于校园助手知识库的公共信息（如培养方案、规章制度、系统功能说明等）。

## 步骤

### Step 1: 使用 Agent Browser 打开登录页面
- URL: https://jwc.sdfmu.edu.cn/academic/common/security/affairLogin.jsp
- 使用 browser_agent 工具打开此页面

### Step 2: 填写登录表单
- 用户名字段 (j_username): [已清理]
- 密码字段 (j_password): [已清理]
- 查看验证码图片，识别其中的字符，填入验证码字段 (j_captcha)
- 点击登录按钮
- 如果验证码识别错误导致登录失败，刷新重试，最多 3 次

### Step 3: 登录成功后，依次浏览以下页面

**必须访问的页面：**
1. 主页/首页 — 记录完整的导航菜单结构
2. 培养方案页面 — 记录学分要求和课程设置信息
3. 学籍信息页面 — 记录页面结构和提供的信息类型
4. 成绩查询页面 — 记录页面提供的功能（不需要记录具体分数）
5. 考试安排页面 — 记录页面结构

**如有时间额外访问：**
6. 选课相关页面
7. 毕业审核页面
8. 教学评价页面
9. 通知公告/规章制度列表页

### Step 4: 对每个页面记录
- 页面标题和 URL
- 页面上的主要功能模块
- 任何规章制度、操作说明、帮助文本的完整内容
- 表单字段和选项列表（帮助理解系统功能）

## 输出
将所有发现写入文件 `kimi/report-7-jwc-browser-explore.md`，包含：
1. 登录状态（成功/失败）
2. 完整导航菜单结构（树形列表）
3. 每个页面的详细内容记录
4. 发现的规章制度/操作指南全文
5. 对知识库建设有价值的信息汇总

## 安全约束
- **只读浏览** — 不点击任何修改、提交、删除按钮
- 不进入修改密码页面
- 报告中不包含具体的个人成绩数据，只记录页面结构和公共信息
