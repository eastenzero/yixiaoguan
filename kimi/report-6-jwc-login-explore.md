## 技术备注

### 系统架构分析
| 项目 | 详情 |
|------|------|
| **技术栈** | Java EE (JSP + Servlet) |
| **Web服务器** | 推测为 Tomcat/WebLogic |
| **前端框架** | 原生JS + jQuery + Layer弹窗 |
| **安全框架** | 疑似基于Acegi Security (Spring Security前身) |
| **加密方式** | MD5 + AES-ECB |

### 登录流程详解
```
1. GET /academic/common/security/affairLogin.jsp
   → 获取登录页面 + randomTag

2. POST /academic/common/security/affairLogin.jsp
   → 提交表单数据:
     - j_username: 学号
     - j_password: AES加密后的密码
     - j_captcha: 验证码
     - randomTag: 防CSRF Token

3. 登录成功 → 重定向到 /academic/index.jsp
   登录失败 → 返回登录页面 + 错误信息
```

### 关键发现
1. **验证码机制**: `j_captcha` 字段为必填，验证码图片URL需进一步分析
2. **密码加密**: 页面引用了 `aes.js` 和 `mode-ecb-min.js`，密码使用AES-ECB加密
3. **CSRF防护**: `randomTag` 隐藏字段用于防止跨站请求伪造
4. **缓存控制**: 响应头设置了 `Pragma: no-cache` 等禁用缓存
5. **X-Frame-Options**: 设置了 `DENY` 防止点击劫持

### URL模式总结
```
/academic/                          # 首页
/academic/common/security/          # 登录相关
/academic/student/query/            # 学生查询功能
/academic/student/arrange/          # 课程安排
/academic/student/evaluate/         # 教学评价
/academic/user/                     # 用户服务
/academic/manager/                  # 管理功能（需特权）
/academic/info/                     # 信息公告
/academic/styles/                   # 静态资源
```
