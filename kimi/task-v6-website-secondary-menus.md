# T3 任务: 探查学校官网二级菜单遗漏

## 任务目标
系统性地探查山东第一医科大学官网 (sdfmu.edu.cn) 的**二级菜单和子站点**，找出之前 3 轮探查中可能遗漏的数据源。

## 背景
前 3 轮探查 (report 1-7) 主要覆盖了一级入口和已知子站点，但用户反馈学校官网还有很多**二级菜单**未被发现。本次任务需要深入每个一级栏目的子菜单和下拉列表。

## 探查范围

### 1. 主站顶部导航栏的所有二级菜单
访问 https://www.sdfmu.edu.cn/，重点检查顶部导航栏每个一级菜单下的二级展开项：

- **学校概况** → 所有子菜单（校史沿革、校训校歌、现任领导、机构设置...）
- **人才培养** → 所有子菜单（本科教育、研究生教育、继续教育...）
- **科学研究** → 所有子菜单（科研平台、科研项目、科研成果...）
- **师资队伍** → 所有子菜单
- **招生就业** → 所有子菜单
- **合作交流** → 所有子菜单
- **校园生活** → 所有子菜单
- **公共服务** → 所有子菜单

### 2. 底部链接和快速入口
检查页面底部、侧边栏的快速链接，可能指向独立子站点。

### 3. 院系/部门子站点
检查以下常见子站点是否存在且未被之前报告覆盖：
- 学生工作处/学生处: xsc.sdfmu.edu.cn 或 xgc.sdfmu.edu.cn
- 后勤管理处: hqglc.sdfmu.edu.cn
- 保卫处: bwc.sdfmu.edu.cn
- 团委: tw.sdfmu.edu.cn
- 研究生院: yjsc.sdfmu.edu.cn 或 yjs.sdfmu.edu.cn
- 国际交流处: gjjlc.sdfmu.edu.cn
- 财务处: cwc.sdfmu.edu.cn
- 教师发展中心
- 实验室管理
- 各二级学院网站

### 4. 与前 3 轮报告对比
已探查的子站点（无需重复）：
