## 重要：必须使用 Agent Browser
本任务**必须使用 browser_agent / Agent Browser 技能**来完成。你需要：

2. 填写账号密码

4. 点击登录按钮
5. 登录成功后逐个访问内部页面

## 登录步骤
1. 用 Agent Browser 打开上述 URL
2. 在用户名输入框填入学号
3. 在密码输入框填入密码（注意：系统前端可能会做 AES 加密，但通过浏览器正常输入应该会自动处理）
4. 查看验证码图片，识别验证码内容，填入验证码输入框
