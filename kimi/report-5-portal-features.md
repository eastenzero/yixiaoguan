## 功能模块详情

### 1. 网上服务大厅
- **入口**: 
  - PC端：学校官网主页(www.sdfmu.edu.cn)右上角"信息门户"→网上服务大厅
  - 移动端：山一大企业微信→工作台→网上服务大厅
- **功能描述**: 一站式"网上服务大厅"平台，将校（院）单位、部门的部分业务由线下办理全面转为线上办理，支持PC端和移动端使用，为师生提供网上申请、事项办理、查询与证明等服务事项网上办，实现"单点登录、全网通办、实时监控"。
- **操作流程**: 
  1. 访问信息门户（首次使用需激活账号）
  2. 登录后进入网上服务大厅
  3. 选择需要办理的服务事项
  4. 填写表单并提交申请
  5. 管理人员在线审批
  6. 查询办理进度和结果
- **服务事项**: 截至2022年已上线18个服务流程，进驻流程近50项，累计服务师生2994人、已完成事项办理12972项
- **常见问题**: 
  - 账号需先激活才能使用
  - 部分流程涉及多部门协同办理
- **联系方式**: 杨老师 0531-59556211（现代教育技术中心）
- **KB价值**: 高
- **信息来源URL**: https://metc.sdfmu.edu.cn/info/1073/1956.htm

### 2. 网上报修
- **入口**: 山一大企业微信→工作台→网上报修
- **功能描述**: 手机端网上报修服务，方便师生提交校园设施设备故障报修申请。
- **操作流程**: 
  1. 打开山一大企业微信
  2. 进入工作台

- **常见问题**:

- **联系方式**: 现代教育技术中心 0531-59556215、0538-6229861

### 3. 接诉即办
- **入口**: 需登录确认（可能位于企业微信工作台或信息门户）
- **功能描述**: 快速响应师生诉求的综合性服务平台，实现"有问必答、接诉即办"，问题收集-解决-反馈闭环式运行。
- **操作流程**: 
  1. 进入接诉即办平台
  2. 提交诉求（可选择实名或匿名）
  3. 系统自动派单至相关部门
  4. 部门处理并反馈
  5. 诉求人查看处理结果并评价
- **可投诉问题**: 涉及教育教学、日常管理、后勤服务等方面的咨询、求助、建议、投诉、举报等
- **处理时限**: 一般诉求24小时内响应，复杂问题多部门协同处理
- **联系方式**: 需登录确认
- **KB价值**: 高
- **信息来源URL**: https://metc.sdfmu.edu.cn/（相关功能页面）

### 9. 人脸采集 & 证件照采集
- **入口**: 山一大企业微信→工作台→人脸采集/证件照采集
- **功能描述**: 学生自助上传和更新人脸照片、证件照，用于校园门禁识别、校园卡等场景。
- **操作流程**: 
  1. 打开山一大企业微信
  2. 进入工作台
  3. 点击"人脸采集"或"证件照采集"
  4. 按提示拍照或上传照片
  5. 提交审核
- **照片要求**: 
  - 可使用手机自拍或从手机图库中选择
  - 不得使用美颜或修图
  - 免冠正面半身照，彩色照片
  - 能看到两耳轮廓和上到头顶下到颈部下端的范围
  - 照片像素不低于640*480，JPG格式
- **注意事项**: 
  - 新生入住前需完成人脸信息采集，否则无法刷开宿舍楼闸机
  - 建议使用近期照片
- **联系方式**: 现代教育技术中心
- **KB价值**: 高
- **信息来源URL**: https://xkc.ujn.edu.cn/info/1025/2248.htm

### 10. 毕业生离校单
- **入口**: 
  - PC端：信息门户→网上服务大厅→热门服务→毕业生离校单
  - 移动端：山一大企业微信→工作台→网上服务大厅→热门服务→毕业生离校单
- **功能描述**: 毕业生离校手续线上办理与提醒服务功能，实现毕业生离校手续办理情况的线上查询和确认。
- **操作流程**: 
  1. 登录网上服务大厅
  2. 选择"毕业生离校单"
  3. 选择学生类别、政治面貌等字段
  4. 确认各环节是否已办理
  5. 勾选"已确认"按钮
  6. 正式提交
- **办理环节**: 包含财务费用清算、党团关系转接、学生档案转移、户口迁移、图书借阅清缴、查体、校园卡注销及退费等八个环节
- **注意事项**: 
  - 如有欠费或未归还图书，无法完成提交
  - 辅导员可查看提交情况并进行催办
  - 催办消息通过统一消息平台推送
- **开通时间**: 每年6月下旬（以2023年为例：6月26日上午9:00）
- **联系方式**: 
  - 现代教育技术中心：杨老师 0531-59556211（技术咨询）
  - 教务部：刘老师 0531-59556806
- **KB价值**: 高
- **信息来源URL**: https://www.sdfmu.edu.cn/info/1074/25276.htm

### 11. 信息门户密码与账号
- **入口**: 学校官网主页(www.sdfmu.edu.cn)右上角"信息门户"
- **功能描述**: 统一身份认证平台，为师生提供统一的账号和密码登录校内各业务系统。
- **账号激活流程**: 
  1. 访问信息门户登录页
  2. 点击"立即激活"
  3. 输入学号/工号和身份证号验证身份
  4. 绑定手机号和邮箱
  5. 设置密码（需包含大小写字母、数字、特殊字符）
  6. 完成激活
- **密码重置**: 
  - 方式一：登录页点击"忘记密码"，通过手机号或邮箱验证重置
  - 方式二：携带身份证到现代教育技术中心服务窗口人工重置
- **注意事项**: 
  - 首次使用必须先激活账号
  - 建议绑定手机号以便找回密码
  - 账号为学号（学生）或工号（教职工）
- **联系方式**: 现代教育技术中心 0531-59556211
- **KB价值**: 高
- **信息来源URL**: https://www.sdfmu.edu.cn/info/1074/25276.htm

## 智慧校园建设概况

山东第一医科大学制定了建设"一网一云一平台，搭建四大应用，构建一个门户，一个中心"的智慧校园建设总体规划。截至相关报道时间：

- 数据中台已汇聚整理教职工、学生、一卡通等主题库17个，数据表492个
- 汇聚教务、科研、研究生等15个系统的数据
- 向11个业务部门的25个业务系统提供了82个数据接口服务
- 网上服务大厅开发线上事项办理流程79个
- 丰富企业微信移动端应用功能，开发学校预约中心，以及网上110、人像与证件照自助采集、空闲教室查询、教师课表查询等服务应用场景
