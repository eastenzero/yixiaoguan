## 发现的问题

### 1. 动态验证码阻碍自动化测试
**严重程度**: 中等
**描述**: 登录页面使用动态生成的数学验证码，在自动化测试过程中频繁刷新，导致难以同步填写正确的验证码值。
**建议**: 
- 为测试环境提供禁用验证码的配置选项
- 或使用固定验证码用于自动化测试
- 提供测试专用绕过认证的接口

### 2. 静态数据展示
**严重程度**: 低
**描述**: 从代码分析可见，当前知识库页面使用的是静态mock数据（knowledgeList数组），而非从后端API动态加载。
**代码位置**: `apps/teacher-web/src/views/KnowledgeView.vue`
**建议**: 完成与后端API的集成，实现数据的动态加载。
