## 问题与建议

### 发现的问题

1. **后端服务未启动**：登录请求无法完成，需要启动 Java 后端服务
2. **验证码动态变化**：每次刷新验证码都会改变，需要实时 OCR 识别或计算
3. **权限控制正常**：未登录用户无法访问聊天页面，安全机制有效

### 建议

1. **启动后端服务后重新测试**：
   ```bash
   # 在后端项目目录执行
   java -jar target/yixiaoguan.jar
   # 或
   ./mvnw spring-boot:run
   ```

2. **使用已登录的 session 进行测试**：
   - 先手动登录获取 token
   - 使用保存的 cookie/token 进行自动化测试

3. **改进验证码处理**：
   - 在测试环境中可以临时关闭验证码
   - 或使用固定的测试验证码
