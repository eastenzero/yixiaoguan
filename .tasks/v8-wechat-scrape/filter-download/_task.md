# FILTER-DOWNLOAD — 筛选并下载有价值文章

**所属**: v8-wechat-scrape / PHASE 2
**优先级**: P0
**执行者**: Kimi
**预估工时**: 3h
**依赖**: SYNC-ARTICLES

## 任务目标

对同步完成的文章列表，按时间范围和关键词筛选，批量下载 Markdown 格式文章。

## 筛选条件

**时间范围**: 2024-04-01 ~ 2026-04-10

**纳入关键词**（标题含任一即下载）:
```
通知 / 公告 / 安排 / 须知 / 指南 / 攻略 / 流程 / 办理
报名 / 缴费 / 政策 / 规定 / 奖学金 / 助学金 / 选课 / 考试
成绩 / 宿舍 / 校园卡 / 图书馆 / 网络 / VPN / 毕业 / 就业
考研 / 保研 / 实习 / 社团 / 竞赛 / 志愿 / 心理 / 讲座
```

**排除关键词**（标题含任一则跳过）:
```
招聘 / 聘用 / 人才引进 / 论文发表 / SCI / 学术会议
领导调研 / 视察 / 座谈 / 转发 / 转载
```

## 下载方式

### 方式一: 自动化 Python 脚本（推荐）

编写并运行 `scripts/wechat/download_articles.py`:

```python
# 伪代码逻辑
1. 读取 wechat-accounts-list.md 中的 A/B 类 fakeid
2. 从已同步的文章列表中按上述条件筛选
3. 对每篇符合条件的文章调用:
   GET /api/public/v1/download?url={article_url}&format=markdown
4. 保存到 wechat-exports/{公众号名称}/{日期}-{标题}.md
```

### 方式二: Web UI 操作

1. 打开 http://localhost:3000 → 文章下载
2. 选择公众号 → 设置时间范围 → 按关键词筛选
3. 全选符合条件的文章 → 点击【抓取】
4. 导出 → 选择 Markdown 格式
