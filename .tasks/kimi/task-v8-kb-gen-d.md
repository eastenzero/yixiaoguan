## 生成规则

1. **一篇文章 → 0~3 条 KB**
   - 内容聚焦的短文章 → 1 条
   - 包含多个独立信息点 → 拆分为多条
   - 纯新闻无实用信息 → 0 条

2. **内容处理**:
   - 提取事实性信息（时间/地点/流程/要求/联系方式）
   - 去除时效性过强表述（"本学期""今年" → 具体日期）
   - 去除营销语句/表情符号引用
   - 去除图片链接

3. **去重规则**:
   - 与 v6/v7 已有 259 条 KB + 批次A/B/C 新 KB 对比主题
   - 同主题取信息量最大或最新的版本
   - 公众号有多篇同主题文章时，取最新一篇

4. **分类建议**:
   - B01/B04 → 学校管理类
   - B02 → 招生入学类
   - B03 → 学生活动类
   - B05 → 科研创新类
   - B06 → 生活服务类
   - B07 → 国际交流类
   - B08 → 财务缴费类
