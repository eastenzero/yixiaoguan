# 考试纪律与违纪处理

## 考核方式

所有课程均须进行考核。考核可采用：
- 笔试
- 口试
- 实际操作
- 提交论文（报告）等不同方式

## 成绩评定

- 考核成绩采用百分制
- 考核应注重学习过程管理，采用形成性评价和终结性评价相结合的方式
- 平时的提问、课堂讨论、作业、测验、实验、考勤情况、见习、期中考核等成绩均可纳入考核结果
- 平时成绩一般占课程总成绩的20%~50%，具体比例由教研室根据课程性质自行确定

## 考试纪律要求

### 考前要求

1. 学生应当按时参加教育教学计划规定的活动
2. 凡缺课累计达到该课程教学时数**三分之一**者，取消其期末考试和补考资格，成绩按零分计
3. 平时作业、实验报告缺交次数累计达到**三分之一**者，取消其期末考试和补考资格，成绩按零分计

### 考中要求

1. 学生应严格遵守考场纪律，服从监考人员管理
2. 按时进入考场，按指定位置就座
3. 携带有效证件（学生证、身份证等）
4. 除考试规定可携带的物品外，其他物品按要求放置
5. 保持考场安静，不得交头接耳、左顾右盼
6. 独立完成答卷，不得抄袭、传递纸条等

## 违纪行为及处理

### 违纪情形

学生严重违反考核纪律或者作弊的，该课程考核成绩记为无效，并应视其违纪或者作弊情节，给予相应的纪律处分。

### 违纪处理

给予**警告、严重警告、记过及留校察看处分**的，经教育表现较好，可以对该课程给予补考或者重修机会。

### 开除学籍的考试作弊情形

学生有下列情形之一，学校可以给予开除学籍处分：

1. 代替他人或者让他人代替自己参加考试
2. 组织作弊
3. 使用通讯设备或其他器材作弊
4. 向他人出售考试试题或答案牟取利益
5. 其他严重作弊或扰乱考试秩序行为的

### 成绩记载

学生考核成绩记入本人档案：

| 情况 | 成绩记载 |
|------|----------|
| 正常考核 | 按实际成绩记载 |
| 重修 | 按实际成绩记载 |
| 补考合格 | 成绩按60分记载 |
| 缓考 | 成绩按实际得分记载 |
| 免修 | 以申请免修考核成绩记载 |
| 旷考 | 该课程以零分记，不允许参加补考 |
| 作弊 | 该课程以零分记，不允许参加补考 |

### 绩点计算

| 情况 | 绩点计算 |
|------|----------|
| 旷考 | 课程绩点为0 |
| 补考不合格 | 课程绩点为0 |
| 补考合格 | 课程绩点为1 |
| 重修、缓考和免修 | 参照实际成绩计算课程绩点 |

## 补考与缓考

### 补考

- 对考核不及格的课程，学校给予**1次补考机会**
- 补考形式及难度与正常考核一致
- 补考仍不合格的，必修课应按规定重修，选修课可重修或另选
- 已满足毕业要求的也可放弃考核不合格的选修课

### 缓考

- 学生因故或其它不可抗拒的因素，不能参加课程正常考核，须在**考试前申请缓考**（因病缓考须有医院证明）
- 经批准可办理缓考手续
- 其后按照学校统一规定时间参加缓考考试

### 旷考

- 学生无故不参加课程正常考核视为旷考
- 旷考学生不得参加相关课程的补考或缓考
- 要取得该课程学分须重修

## 诚信教育

学校应当开展学生诚信教育，以适当方式记录学生学业、学术、品行等方面的诚信信息，建立对失信行为的约束和惩戒机制。对有严重失信行为的，可以规定给予相应的纪律处分，对违背学术诚信的，可以对其获得学位及学术称号、荣誉等作出限制。

## 注意事项

- 学生应认真复习，诚信应考
- 考试前应了解考试安排，带齐证件
- 考试违纪将影响个人档案记录
- 作弊行为将受到严肃处理，甚至开除学籍
- 对处分决定有异议的，可在规定期限内提出申诉
