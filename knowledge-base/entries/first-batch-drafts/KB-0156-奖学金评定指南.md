# 奖学金评定指南

## 奖学金类别

### 1. 国家奖学金

- **金额**：10000元
- **发放时间**：秋季学期（预计12月底）
- **名额**：全校共64人，学校差额评选，报送省里再次进行差额评选
- **对象**：二年级以上（含二年级）全日制本科学生

### 2. 省政府奖学金

- **金额**：6000元
- **发放时间**：秋季学期（预计12月底）
- **名额**：全校共30人，学校差额评选，报送省资助中心27人
- **对象**：二年级以上（含二年级）全日制本科学生

### 3. 国家励志奖学金

- **金额**：6000元
- **发放时间**：秋季学期（预计12月底）
- **对象**：全日制二年级及以上本科学生，2024-2025学年已认定的家庭经济困难学生
- **要求**：学习成绩排名或综合考评成绩排名有一项位于前30%，另一项位于前50%

### 4. 省政府励志奖学金

- **金额**：5000元
- **发放时间**：秋季学期（预计12月底）
- **对象**：全日制二年级及以上本科学生，家庭经济困难学生

### 5. 校级综合奖学金

| 等级 | 金额 | 评选比例 |
|------|------|----------|
| 一等奖学金 | 1500元/学年 | 参评学生总数的4% |
| 二等奖学金 | 1000元/学年 | 参评学生总数的4% |
| 三等奖学金 | 800元/学年 | 参评学生总数的10% |

## 互斥规定

同一学年内，本科生**国家奖学金、省政府奖学金、国家励志奖学金和省政府励志奖学金具有互斥性**，学生只能获得其中一项。

## 基本评选条件

### 通用条件

1. 热爱社会主义祖国，拥护中国共产党的领导
2. 遵守宪法和法律，遵守学校规章制度，无违纪现象
3. 诚实守信，道德品质优良
4. 学习态度端正，目的明确，刻苦努力

### 成绩要求

**国家奖学金/省政府奖学金**：
- 学习成绩优异，社会实践、创新能力、综合素质等方面特别突出
- 以2024-2025学年的获奖情况和综合表现为主

**励志奖学金**：
- 2024-2025学年没有不及格科目（含V类课程），无不及格现象
- 认定为困难级别的，学习成绩排名或综合考评成绩排名有一项位于前30%，另一项位于前50%
- 认定为特殊困难级别的，学习成绩排名和综合考评成绩排名可放宽至前50%

**校级综合奖学金**：
- 学习成绩排名在评选范围（自然班）内位于前30%
- 2024-2025学年没有不及格科目（含V类课程）或补考或过程性考核不达标等现象
- 综合测评成绩排名在评选范围（自然班）内位于前40%

## 评审工作流程

### 国家奖学金/省政府奖学金

1. **学生申请**：符合条件的学生个人提出申请，辅导员初步审核
2. **学院评审**：学院组织公开答辩评议会，由学院奖助学金评审委员会全体成员进行投票，确定推荐名单
3. **学院公示**：在学院进行不少于2天的公示
4. **学校评审**：校（院）本科生奖助学金评审委员会进行差额评选
5. **学校公示**：在全校范围内进行5个工作日的公示
6. **上报审批**：公示后报送省资助中心审核

### 励志奖学金

1. **学生申请**：符合评审条件的学生提出申请
2. **班级评议**：经过班级评议、辅导员初步审核，提交学院评审委员会
3. **学院评审**：学院评审委员会通过民主评议等方式确定拟推荐名单并进行排序
4. **学院公示**：在学院内进行不少于2天的公示
5. **学校评审**：校（院）评审委员会进行审核和差额评审
6. **学校公示**：在全校范围内进行5个工作日的公示
7. **上报审批**：公示后报送省资助中心审核

### 校级综合奖学金

1. 个人申请
2. 班级评议
3. 学院审核
4. 学院内公示不少于2个工作日
5. 报送学生资助管理中心

## 材料报送要求

### 国家奖学金材料

1. 获奖学生初审名单表（电子版和纸质版，加盖学院公章）
2. 申请审批表（WORD和EXCEL版）
3. 候选人情况统计表
4. 学生成绩单
5. 成绩和综合测评排名表
6. 获奖证书扫描件和复印件
7. 学院奖助学金评审委员会名单、公示照片

### 时间节点

- **国家奖学金**：9月25日前报送
- **省政府奖学金**：11月前报送
- **励志奖学金**：10月前报送电子版，公示材料等
- **校级综合奖学金**：12月前报送

## 注意事项

- 各学院要广泛动员学生积极报名，优先推荐家庭经济困难学生参与
- 评审过程必须严格按照文件要求开展，做到公开、公平、公正
- 坚决杜绝暗箱操作、挪用、克扣及平分奖助学金的违纪违规行为
- 学院组织公开答辩评议会时，要创造条件让在校学生观摩学习
