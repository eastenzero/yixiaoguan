# 居民医保停保办理流程

## 一、核心问题与解答

### Q1: 如何办理居民医保停保？
**A:** 办理居民医保停保需通过鲁医保微信小程序，按以下三步操作：

### Q2: 第一步做什么？
**A:** **选择参保城市**
- 在微信小程序中搜索鲁医保
- 点击进入后，在省本级框中选择以前所参保的城市

### Q3: 第二步做什么？
**A:** **完成个人信息注册**
- 进入所在城市的XX医保微信小程序
- 点击我的
- 进入后点击登录/注册
- 按照提示进行个人信息认证

### Q4: 第三步做什么？
**A:** **完成居民医保停保中断操作**
- 完成登录/注册后，点击我要办事
- 选择城乡居民医疗保险中断登记功能
- 按提示完成居民医保停保中断

## 二、办理流程图

```
微信小程序搜索鲁医保
       ↓
选择以前所参保的城市
       ↓
进入XX医保小程序
       ↓
点击我的 → 登录/注册
       ↓
完成个人信息认证
       ↓
点击我要办事
       ↓
选择城乡居民医疗保险中断登记
       ↓
按提示完成停保
```

## 三、注意事项

- 需要先选择以前所参保的城市（非当前所在城市）
- 必须完成实名认证后才能办理停保
- 停保操作在城乡居民医疗保险中断登记功能中

## 四、关键信息

| 项目 | 内容 |
|------|------|
| 办理平台 | 鲁医保微信小程序 |
| 关键功能 | 城乡居民医疗保险中断登记 |
| 必要步骤 | 选择参保城市 → 注册认证 → 办理中断 |
