# 困难认定与助学金申请指南

## 困难认定概述

### 认定对象

家庭经济困难学生是指学生家庭经济能力难以满足在校期间学习、生活基本支出的**全日制本专科学生**。

### 认定标准

参照居民最低生活保障标准，家庭经济困难学生的认定标准设为：

1. **特殊困难**
2. **困难**

### 认定时间

- 每学年开展一次
- 每年**9月启动**
- 次年**4月进行资格复查**

## 认定程序

### 1. 个人申请

申请学生向班级(或年级、专业)评议小组提交：
- 《家庭经济困难学生认定申请表》
- 《家庭经济状况查询核对授权书》
- 个人家庭经济情况说明
- 低保证、建档立卡证等相关材料复印件

### 2. 班级评议

- 班级(或年级、专业)评议小组对学生提交的材料进行民主评议
- 以学生家庭人均收入对照学校确定的家庭经济困难学生认定标准
- 结合学生日常生活、学习表现等确定学生是否困难及困难档次
- 将评议结果在班级(或年级、专业)范围内进行公布（只公布姓名和困难档次）
- 公布无异议后报学生资助工作领导小组

**着重考虑对象**：
- 脱贫享受政策学生
- 防止返贫监测帮扶对象
- 低保家庭学生
- 孤残学生
- 烈士子女
- 家庭成员长期患重病
- 家庭遭遇自然灾害或突发事件等特殊情况的学生

### 3. 学院审核

- 学院学生资助工作小组认真审核评议小组报送的初步评议结果
- 将审核通过的家庭经济困难学生名单及困难档次在学院范围内公示**不少于2个工作日**
- 如有异议，应在接到异议材料的3个工作日内予以答复

### 4. 学校审批

- 学生资助管理中心审核学院报送的评议结果
- 将审核通过的家庭经济困难学生名单及困难档次在学校范围内公示**不少于5个工作日**
- 学校学生资助工作领导小组审核批准家庭经济困难学生名单及档次

## 特殊困难学生认定

### 认定为"特殊困难学生"的情形

1. 脱贫家庭(继续享受政策)学生
2. 脱贫不稳定家庭学生
3. 边缘易致贫家庭学生
4. 低保家庭学生
5. 特困救助供养学生
6. 孤儿
7. 重点困境儿童
8. 烈士子女
9. 家庭经济困难的残疾学生及残疾人子女
10. 因其他原因(如家庭遭受重大自然灾害、家庭成员遭遇突发重大疾病或意外)造成家庭经济特别困难的学生

### 认定为"困难学生"的情形

1. 父母务农或父母一方暂时失业，家庭成员中有疾病且医疗费用负担较重的
2. 单亲家庭且单亲父(母)无经济收入或收入无法维持学生本人学习、生活需要的
3. 父母务农或父母一方暂时失业，有多名子女同时在接受非义务教育阶段的普通学历教育，家庭经济负担较重的
4. 因遭受自然灾害或突发事故家庭财产损失较重的
5. 因其他原因造成家庭经济困难的

### 不能认定为困难学生的情形

1. 隐瞒家庭经济实际情况、提供虚假信息的
2. 由于家庭建房、购房、购车等原因造成家庭经济暂时困难的
3. 由于生活奢侈浪费等原因造成生活暂时困难的
4. 有其他不符合认定条件的

## 国家助学金申请

### 申请条件

1. 热爱社会主义祖国，拥护中国共产党的领导
2. 遵守宪法和法律，遵守学校规章制度、无违纪现象
3. 诚实守信，道德品质优良
4. 学习态度端正，目的明确，刻苦努力，成绩良好
5. 生活上艰苦朴素，注意勤俭节约，无铺张浪费现象
6. 家庭经济困难，已被认定为困难学生

### 申请流程

1. 登录山东省学生资助一网通办系统
2. 填写个人基本信息、家庭基本信息和银行卡信息
3. 填写家庭经济状况
4. 上传申请理由说明及证明材料
5. 上传个人承诺书
6. 上传经济状况核查授权书
7. 完成确认并提交

### 系统登录方式

**PC端**：
- 电脑浏览器访问山东省政务服务网
- 在【教育考试】标题下点击【学生资助一网通办平台】
- 登录山东省统一身份认证平台

**手机端**：
- 在手机应用商店下载爱山东APP
- 在搜索栏搜索【学生资助】
- 点击【山东省学生资助一网通办平台】

### 特殊群体学生

- 名单来源于山东省及全国资助中心
- 登录后无需填表，系统默认为特别困难学生（最终等级以评议结果为准）
- 免申即享相应资助项目

## 银行卡信息

- 学生银行账户建议填写学校配发的银行卡号
- **中国农业银行储蓄卡开户行信息**：
  - 济南校区——中国农业银行济南大舜支行
  - 泰安校区——中国农业银行泰安旅游开发区支行

## 注意事项

- 认定工作必须严格评审条件，规范工作程序，做到公开、公平、公正
- 申请学生应在相应学年家庭经济困难学生档案库中
- 隐瞒家庭经济实际情况、提供虚假信息的，应追回资助金，并根据情节给予批评教育，直至纪律处分
- 学生银行账户信息应准确无误，以免影响资助金发放
