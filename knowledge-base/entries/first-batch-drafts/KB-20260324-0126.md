## 问题概述

本通知部署了2025-2026学年本科生国家助学金评审工作，明确了资助标准、评审范围、申请条件、工作流程及材料报送要求，适用于各学院组织国家助学金评审时使用。

## 标准答复

2025-2026学年国家助学金分三档：一档2600元、二档3700元、三档4800元，秋季学期（预计12月）和春季学期（预计次年5月）各发放一半。评定对象限定为已认定的2025-2026学年家庭经济困难学生；公费生、退役复学学生不参评；超过专业基本修业年限的在校生不再享受国家助学金。

申请学生须道德品质优良、家庭经济困难、生活俭朴、勤奋学习、积极上进。特殊困难类型学生（脱贫家庭、低保、特困救助、孤儿、烈士子女、残疾学生及残疾人子女、突发严重困难家庭学生等）原则上给予不低于二档的资助。建档立卡学生如自愿放弃，须提交本人手写书面放弃声明。

评审流程：召开主题班会 → 学生填表 → 班级评议 → 辅导员初审 → 班级公示 → 学院评审委员会审议并公示不少于2天 → 校（院）评审委员会审核并全校公示5个工作日 → 报送省资助中心。

## 办理流程

1. 学院召开国家助学金评审工作主题班会，组织符合条件学生填写《2025-2026学年国家助学金申请表》。
2. 班级评议、辅导员初审、班级公示后提交学院奖助学金评审委员会。
3. 学院审议确定建议资助名单及档次，并在学院范围公示不少于2天。
4. 校（院）评审委员会审核，确定初步名单及档次，并在全校公示5个工作日。
5. 公示后报送省资助中心审核。
6. 按要求报送电子版及纸质版汇总表、放弃声明、公示材料等。

## 注意事项

- 公费生、退役复学学生不参评；退役士兵国家助学金前期已完成申报，不在本次评审范围内。
- 超过专业基本修业年限的在校生不再享受国家助学金。
- 申请学生必须在相应学年家庭经济困难学生档案库中，不符合条件者直接取消资格，不允许更换人选。
- 严禁暗箱操作、挪用、克扣及平分国家助学金，违者将严肃处理。
- 所有材料由学院汇总后统一报送，不接收学生个人或单个班级报送。电子版发送至 shanyidazzzx@163.com。
- 时间节点：10月24日前完成一网通办系统学院审核；10月27日前报送《汇总表》电子版及纸质版（加盖学院公章）。
