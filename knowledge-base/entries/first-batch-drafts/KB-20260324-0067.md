# 2025外研社·国才杯英语能力大赛校赛通知

## 一、核心问题与解答

### Q1: 外研社·国才杯大赛的主题是什么？
**A:** 本届大赛以理解中国，沟通世界为主题，旨在推动三进工作、培育外语人才，讲好中国故事、服务国际传播。

### Q2: 校赛有哪些分赛项？
**A:** 本届大赛英语组校赛设有四个分赛项：
1. **综合能力**
2. **演讲**
3. **笔译**
4. **口译**

### Q3: 各赛项的比赛形式是什么？
**A:**
- **综合能力、口译和笔译赛项：** 均为线下比赛，分别设有泰安校区和济南校区两个考场
- **演讲赛项：** 英语专业组和大学英语组分别组织初赛、复赛和决赛

### Q4: 参赛对象是谁？
**A:** 山东第一医科大学各教学机构在校学生。

### Q5: 有问题如何联系？
**A:** 有意向参赛的同学仔细查看附件，如有问题，请联系对应赛项的负责老师。
