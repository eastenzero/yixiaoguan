# 出国留学指南

## 对外合作交流部概况

校（院）对外合作交流部与外国语学院共同发起的"一带一路"对外交流服务中心，积极响应国家倡议，主动适应校（院）发展要求。

### 主要业务

**培训板块**：
- 综合语言（英语、俄语）培训
- 雅思/托福考试培训
- 语言预科等培训

**出国留学项目板块**：
- 在校生短期出国交流
- 在校生出国读研
- 在校教师攻读境外高校博士学位
- 高中毕业生赴国外留学

## 申请国家公派出国留学的外语条件

### 高级研究学者、访问学者及博士后类别

需达到以下条件之一：

1. **全国外语水平考试(WSK)**：
   - 英语（PETS5）：笔试总分55分（含）以上，其中听力部分18分（含）以上，口试总分3分（含）以上
   - 德语(NTD)：笔试总分65分（含）以上
   - 法语(TNF)：笔试总分60分（含）以上
   - 日语（NNS）/俄语：笔试总分60分（含）以上，其中口试总分3分（含）以上

2. **外语专业本科（含）以上毕业**（专业语种应与留学目的国使用的语种一致）

3. **留学经历**：近十年内曾在同一语种国家或地区连续留学8个月（含）以上，或连续工作12个月（含）以上

4. **出国留学人员培训部**：参加相应语种培训并获结业证书（英语为高级班，其他语种为中级班）

5. **雅思托福等**：
   - 雅思（学术类）6.5分
   - 托福网考95分
   - 德语、法语、西班牙语、意大利语达到欧洲统一语言参考框架（CECRL）B2级
   - 日语达到日本语能力测试（JLPT）三级（N3）
   - 韩语达到TOPIK3级

### 博士研究生、联合培养博士研究生、硕士研究生及联合培养硕士研究生类别

外语水平需达到以下条件之一：

1. 外语专业本科（含）以上毕业
2. 近十年内曾在同一语种国家留学一学年（8-12个月）或连续工作一年（含）以上
3. 参加"全国外语水平考试"（WSK）并达到合格标准
4. 雅思6.5分，托福（IBT）95分，德、法、意、西语达到CECRL的B2级，日语达到二级（N2），韩语达到TOPIK4级
5. 在教育部指定出国留学培训部参加相关语种培训并获得结业证书
6. 通过国外拟留学单位组织的面试、考试等并达到其入学语言要求

### 本科插班生类别

外语水平需达到以下条件之一：

1. 外语专业在读本科二年级（含）以上学生
2. 曾在同一语种国家留学一学年（8-12个月）以上
3. 参加"全国外语水平考试（WSK）"并达到合格标准
4. 在教育部指定出国留学培训部参加相关语种培训并获得结业证书
5. 雅思6.5分，托福95分，德、法、意、西语达到CECRL的B2级，日语达到二级（N2），韩语达到TOPIK4级
6. 通过国外拟留学单位组织的面试或笔试达到其语言要求

### 外语成绩有效期

全国外语水平考试（WSK）、教育部出国留学人员培训部结业证书、雅思、托福、欧洲统一语言参考框架（CECRL）、韩语（TOPIK）、日语（JLPT）成绩有效期均为**两年**。

## 出国留学流程

1. **学生申请**
   - 登录学校国际和台港澳事务管理系统（https://ws.sdfmu.edu.cn/login）
   - 按项目要求填写报名信息
   - 提交外语成绩证明、家长知情同意书等材料
   - 在通知指定截止时间前完成提交

2. **校内审核**
   - 学校对外合作交流部对报名材料进行审核
   - 项目主办方（外方/合作机构）确定最终录取结果

3. **行前准备**
   - 完成校内审批手续
   - 与合作方签署正式协议
   - 购买境外期间保险
   - 按项目要求做好行前培训

4. **后续认定**
   - 访学/留学结束后，本科生可按《山东第一医科大学（山东省医学科学院）"第二课堂成绩单"实施办法》申请第二课堂积分
   - 符合条件的可申请学校留学奖学金

## GPA计算方式

我校GPA计算方式：

```
课程绩点 = (课程成绩÷10) - 5
课程成绩不足60分的，课程绩点为0
课程学分绩点 = 课程绩点 × 课程学分数
平均学分绩点(GPA) = 课程学分绩点之和 ÷ 课程学分之和
```

## 注意事项

- 出国留学应提前规划，准备外语考试
- 注意外语成绩的有效期
- 申请时应如实填写信息，提供真实材料
- 留学期间应遵守当地法律法规和校规校纪
- 保持与学校的联系，按时完成学业
