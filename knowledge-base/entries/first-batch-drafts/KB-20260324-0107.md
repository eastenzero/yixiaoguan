# 免予执行《国家学生体质健康标准》申请流程

## 一、核心问题与解答

### Q1: 什么情况下可以申请免予执行《国家学生体质健康标准》？
**A:** 因身体等特殊原因不能参加体育锻炼的学生，可以申请免予执行《国家学生体质健康标准》，经批准后可修保健课，成绩以保健课成绩计入体美劳育测评。

### Q2: 申请免测需要填写哪些信息？
**A:** 申请表需填写以下信息：
- 基本信息：姓名、性别、学号、年级、专业班级、民族、出生日期
- 申请原因：详细说明不能参加体育锻炼的身体原因
- 申请人签字及日期

### Q3: 申请免测需要哪些审批流程？
**A:** 审批流程包括：
1. **体育教师签字**（无体育课学生可由辅导员老师签字）
2. **家长签字**（可由学生本人代签）
3. **学校体育部门意见**（学校签章及日期）

## 二、申请流程

```
学生填写申请表 → 说明申请原因 → 体育教师/辅导员签字 → 家长签字 → 学校体育部门审核签章
```

## 三、注意事项

- 申请原因需详细说明身体状况
- 无体育课学生，"体育教师签字"一栏可由辅导员老师签字
- "家长签字"一栏可由学生本人代签
- 最终需经学校体育部门审核签章方为有效
