# 医药管理学院学生综合素质测评办法

## 一、核心问题与解答

### Q1: 综合素质测评什么时候进行？多久一次？
**A:** 每学年进行一次，一般于新学年开学后一个月内完成。

### Q2: 综合素质测评成绩由哪几部分组成？各占多少比例？
**A:** 综合素质测评成绩由四部分组成：
- 德育测评：10%
- 学业考核：70%
- 体美劳育测评：10%
- 拓展性素质测评：10%

**计算公式：** 综合素质测评成绩 = 德育测评成绩×10% + 学业考核成绩×70% + 体美劳育测评成绩×10% + 拓展性素质测评成绩×10%

### Q3: 学业考核成绩如何计算？
**A:** 学业考核根据一学年内所修课程初次成绩计算：
- **必修课**占学业考核成绩的60%
- **限定选修课**占学业考核成绩的40%

**计算方法：**（单科分数×学分，所有科目相加）÷总学分

**示例：** 必修1（80分，2学分）、必修2（90分，1学分）、必修3（70分，3学分）
- 必修总成绩：【（80×2+90+70×3）÷6】×60%

### Q4: 体美劳育测评包括哪些内容？
**A:** 
- **体育测评：** 开设体育课的年级按课程成绩计入；未开设体育课的年级按《国家学生体质健康标准》成绩计入；修保健课的同学以保健课成绩计入
- **美育测评：** 开设美育课的年级按课程成绩计入；未开设的年级不计算成绩
- **劳育测评：** 开设《劳动教育》课程的年级，当学年获得必选0.5学分的按课程成绩计分；未获得当学年劳动必选学分的计为0分；未开设的年级不计算成绩

**计算公式：** 体美劳育测评成绩 = 体、美、劳育课程成绩总和 ÷ 课程门数

### Q5: 拓展性素质测评包括哪些内容？满分多少？
**A:** 拓展性素质测评主要包括四项指标，满分10分：
1. **学术与创新**
2. **实践与服务**
3. **社会工作**
4. **文体活动**

**计算方式：** 纪实考核分 + 获奖情况分（同一指标同一项目获奖按最高项加分，不重复加分）

### Q6: 拓展性素质测评获奖加分标准是什么？
**A:** 

| 指标 | 项目 | 加分标准 |
|------|------|----------|
| 学术与创新 | 创新创业及学科竞赛获奖 | 国家级6分，省级4分，市级3分，校级2分 |
| | 主持或参与科研课题、科研成果 | 国家级5分，省级4分，市级2分，校级1分 |
| | 发表论文 | SCI/EI/CSSCI 6分，国内核心B类及以上4分，C类2分，省市级一般期刊1分 |
| | 获得专利 | 发明专利3分，实用新型专利2分，软件著作权1分，外观设计专利1分 |
| | 通过英语、计算机考试 | CET四级2分、六级4分；计算机三级1分、二级3分 |
| 实践与服务 | 突发事件突出表现 | 每项1-3分；为学校赢得重大荣誉可加至6分 |
| | 社会实践获得奖励 | 国家级5分，省级3分，市级2分，校级1分 |
| 社会工作 | 担任学生干部 | 校级组织主席6分，部长5分，干事3分；院级组织主席/志愿服务队队长5分，部长4分，干事2分；班级班长团支4分，其他班委2分，舍长1分 |
| 文体活动 | 参加文体活动竞赛获奖 | 国家级5分，省级4分，市级3分，校级2分，学院1分 |
| 其他 | 获得各级各类荣誉 | 国家级5分，省级4分，市级3分，校级2分，学院1分 |

**备注：**
1. 奖项分等次的，同级别依次递减0.25分；优秀奖、参与奖可按下一级别最高分计分
2. 多人项目有排名的，按位次顺序递减0.2分
3. 项目累计加分最高者按满分计算，其他按比例折算

### Q7: 哪些情况会被扣分？
**A:**
- 受记过及以上处分者：扣5分
- 受记过以下处分者（含通报批评、警告、严重警告）：扣1-3分
- 受到学业预警者：扣2分
- 综合素质测评中弄虚作假者：扣5分

### Q8: 什么情况下不能参加荣誉称号评选？
**A:** 思想道德素质测评成绩在班级后20%者，不得参加本学年任何荣誉称号的评选。

### Q9: 测评工作程序是什么？
**A:**
1. **班级测评：** 班级测评小组按照测评办法进行测评，辅导员审核真实性
2. **年级排名：** 计算综合素质测评成绩，在班级内公示，并进行年级/专业/班级排名
3. **学院审核：** 学院综合素质测评工作领导小组审核，在学院内公示3个工作日
4. **报备：** 公示无异议后，报学生工作部备案，原始材料整理归档

### Q10: 对测评结果有异议怎么办？
**A:** 若学生对测评过程、结果存有疑义，可在公示期内向辅导员、学院综合素质测评工作领导小组或学生工作部反映。如发现弄虚作假现象，将取消相关学生评奖选优资格，并追究相关人员的责任。

## 二、规则提炼

### R1: 德育测评评价标准（100分）

| 一级指标 | 二级指标 | 三级指标要点 |
|----------|----------|--------------|
| 思想政治（30分） | 理想信念（10分） | 拥护党的领导；青年大学习每学年学习率达90%（1年只能漏做1次） |
| | 道德品行（10分） | 践行社会主义核心价值观，遵守公民道德规范 |
| | 态度行为（10分） | 具有四个意识；积极参加政治学习和思想理论教育活动 |
| 遵规守纪（25分） | 法律意识（10分） | 法治观念强，知法懂法，拒绝黄赌毒 |
| | 纪律观念（10分） | 无旷课、旷操、迟到、早退；班会出勤旷一次扣2分，累计三次以上记0分 |
| | 安全意识（5分） | 防火防盗防传染疾病；抵制不良网贷、防范电信诈骗 |
| 身心健康（15分） | 身体素质（5分） | 积极参加体育活动 |
| | 心理素质（5分） | 积极自信，良好情绪管理和压力管理能力 |
| | 人际交往（5分） | 与同学、老师和家人和谐相处 |
| 日常行为（30分） | 文明养成（10分） | 遵守大学生行为规范，文明就餐上网 |
| | 集体观念（10分） | 参加班级集体活动低于平均数，少1次扣2分，少3次以上记0分 |
| | 育人实践（10分） | 参加社会实践、志愿服务、公益活动和勤工助学 |
