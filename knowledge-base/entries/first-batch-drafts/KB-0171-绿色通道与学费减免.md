# 绿色通道与学费减免

## 绿色通道

### 定义

"绿色通道"，是指允许家庭经济特别困难的学生在缓交全部或部分学费、住宿费的情况下先办理入学手续，是确保家庭经济特别困难新生顺利入学的有效措施。

### 适用对象

校（院）正式录取的全日制本专科班学生。

### 申请条件

符合以下条件之一的新生可申请绿色通道：

1. **建档立卡学生**
   - 不在全国和山东省某门出具的证明（需说明：此处原文可能有误，应为提供相关证明）

2. **特殊群体学生**
   - 家庭经济困难的烈士子女
   - 残疾学生
   - 残疾人员子女
   - 低保家庭子女
   - 孤儿
   - 需提供相应证件

3. **家庭重大困难**
   - 家庭主要成员患重病且丧失劳动能力，致使家庭经济特别困难者

4. **不可抗力因素**
   - 家庭遭受重大自然灾害或其它一些不可抗力因素的影响，使家庭财产价值不能保障

### 申请及审批程序

1. **提交申请**
   - 家庭经济困难新生凭《新生入学通知书》、《学生及家庭情况调查表》等证明材料，向所在学院提出申请

2. **学院初审**
   - 各学院经初步认定后，指导新生填写《山东第一医科大学（山东省医学科学院）新生入学"绿色通道"申请表》一式两份
   - 由学生工作办公室和学院负责人签署意见

3. **备案登记**
   - 申请学生应填写《山东第一医科大学（山东省医学科学院）新生入学"绿色通道"申请表》
   - 报学生资助管理中心备案

4. **办理入学**
   - 学生可凭申请表先办理其它入学手续，入住学生公寓
   - 并获得一定额度的新生入学资助

### 后续资助

学生入学后，校（院）将根据《山东第一医科大学（山东省医学科学院）家庭经济困难学生认定办法》对学生进行困难认定，按照有关政策给予资助。

### 诚信要求

学生必须如实申报家庭经济状况和生源地获得资助的情况，**一经发现虚假瞒报，校（院）将视情节给予严肃处理**。

## 学费减免

### 总则

学费减免用于资助确有特殊困难的全日制本专科学生，帮助他们顺利完成学业。

### 减免标准

- 一般按照学生实际学年学费的**30%-100%**减免
- 最高不超过**8000元/人/年**
- 特殊情况需经校(院)领导批准

### 申请条件

符合以下条件之一的学生可申请学费减免：

1. 孤儿
2. 烈士子女
3. 学生家庭或学生本人发生重大意外事故等家庭经济特殊困难学生

### 不得申请的情况

有下列情况之一者**不得申请**学费减免：

1. 因考试作弊或其他违纪行为受到上级有关部门或学校处分者
2. 符合国家生源地信用助学贷款申请条件，无故不愿申请者

### 审批程序

1. **提交申请**
   - 学生应在每学年秋季学期学分确认后，向本学院提出减免当学年学费的申请
   - 提供相关的证明材料
   - 填写《学费减免审批表》

2. **审核审批**
   - 报学生资助管理中心审核
   - 经学校批准后，计划财务部监督执行

### 孤儿学生保障工作

根据《山东省人民政府办公厅关于加强孤儿保障工作的意见》（鲁政办发〔2011〕47号）文件精神：

- 学校**免收孤儿学费、住宿费**
- 每年新生入学后，由学生资助管理中心统计确定孤儿学生名单
- 经学校批准后，由计划财务部监督执行

### 违规行为处理

隐瞒家庭经济实际情况、提供虚假信息的：
- 应追回资助金
- 并根据情节给予批评教育，直至纪律处分

## 组织保障

### 领导机构

校（院）学生资助工作领导小组全面领导"绿色通道"和学费减免工作。

### 执行机构

学生资助管理中心具体负责组织和管理工作，各学院负责审核、审批工作。

### 服务要求

各学院应本着实事求是的原则，对家庭经济困难学生提供积极、主动、热情的服务，除允许其缓交相关费用先行办理入学手续外，还要指导、帮助学生解决生活中存在的困难。

## 注意事项

- 绿色通道仅限家庭经济特别困难的新生入学时使用
- 学费减免应在规定时间内申请
- 所有申请材料应真实有效
- 获得资助后应努力学习，回报社会
- 家庭经济状况好转后应主动告知学校
