# 2025年宪法卫士行动计划参与指南

## 一、核心问题与解答

### Q1: 什么是宪法卫士行动计划？
**A:** 宪法卫士行动计划是第十届全国学生学宪法讲宪法活动的一部分，由教育部主办，学生通过教育部青少年普法网完成在线学习。

### Q2: 谁需要参加宪法卫士行动计划？
**A:** 校（院）组织学生参加2025年宪法卫士行动计划，具体由各学院辅导员按照分配的班级管理员组织学生参与。

### Q3: 如何登录教育部青少年普法网？
**A:**
- **网址：** https://qspfw.moe.gov.cn
- **登录方式：**
  - 去年注册的账号可继续使用
  - 如忘记密码，可登录页面点击【找回密码】自行找回
  - 新增班级管理员初次登录时，使用附件1中的密码，登录后可自行修改

### Q4: 班级管理员需要做什么？
**A:** 班级管理员登录后需要：
1. 按《操作指南》添加各自所带的每名学生进入各自班级
2. 发动学生根据普法网第十届全国学生学宪法讲宪法活动专题页面发布的2025年宪法卫士行动计划的活动流程与操作指南，完成在线学习

### Q5: 学生需要完成什么任务？
**A:** 学生需要：
1. 被班级管理员添加进班级
2. 根据普法网专题页面发布的活动流程与操作指南
3. 完成在线学习

## 二、操作流程

```
辅导员分配班级管理员 → 班级管理员登录普法网 → 添加学生进入班级 → 发动学生完成在线学习 → 学生按要求完成学习任务
```
