# 学生证件管理

## 学生证补办

### 补办流程

学生证补办须按以下流程办理：

1. **以学院为单位办理**
   - 不接受个人的办理申请
   - 不补办乘车优惠卡

2. **缴纳工本费**
   - 各学院以本学院为单位，到财务收费科交纳工本费（**6元/人**）
   - 咨询电话：0531-59556889、0538-6229985

3. **提交材料**
   - 携带《山东第一医科大学补办学生证审批汇总表》（附件1）
   - 财务缴费票据
   - 统一补办

### 注意事项

- 学生证是重要的学生身份证明，应妥善保管
- 学生证遗失应及时补办
- 不得转借、涂改学生证

## 团员组织关系转接

### 团员组织关系介绍信模板

**团员介绍信存根**：
> ____同志系中国共产主义青年团团员，组织关系由____转到____。

**中国共产主义青年团团员组织关系介绍信**：
> __________同志（男/女），________岁，________族，系中国共产主义青年团团员，身份证号码________，由________去________，请转接组织关系。该同志团费已交到2022年6月。

**回执联**：
> ______________同志的团员组织关系已转达我处，特此回复。

### 转接说明

- 回执联由接收团员组织关系的基层团委在接收团员后一个月内邮寄或传真至团员原所在基层团委
- 学院自行编号、留存存根

## 入团入党申请

### 入团申请

**基本格式**：
1. **标题**：入团申请书（第一行正中书写）
2. **称呼**：敬爱的团组织（顶格写，后面加冒号）
3. **正文**：
   - 介绍个人的现实情况
   - 入团动机和理由
   - 对团的认识和决心
4. **结尾**：恳请团组织早日批准我加入中国共产主义青年团、此致敬礼
5. **落款**：申请人姓名和成文日期（右下角）

**要求**：
- 使用信纸书写
- 内容不少于3页纸
- 字体端正，不要有涂改和错字

### 入党申请

**基本格式**：
1. **标题**：入党申请书（居中写）
2. **称谓**：敬爱的党组织（顶格书写，后面加冒号）
3. **正文**：
   - 对党的认识、入党动机和对待入党的态度
   - 个人在政治、思想、学习、工作等方面的主要表现情况
   - 今后努力方向以及如何以实际行动争取入党
4. **结尾**：请党组织在实践中考验我、此致敬礼
5. **落款**：申请人XXX，XXXX年X月X日（居右书写）

**注意事项**：
- 要认真学习党章，掌握基本精神
- 联系自己的思想实际谈对党的认识和入党动机
- 对党要忠诚老实，向党组织反映真实思想情况
- 申请书要写得朴实、庄重
- 由申请入党的同志亲笔书写，亲自交给党组织
