# 校外住宿管理规定

## 适用范围

本规定适用于我校（院）全日制在籍研究生、本科生、专科生。

## 校外住宿定义

学生校外住宿是指学生在校（院）或实践教学基地安排的学生宿舍外的场所（包括校内其他住房）租（借）公私房屋住宿。

## 基本原则

- 校（院）**原则上不允许**学生私自走读、外宿或租房住宿等任何形式的校外住宿
- 学生均应在校（院）安排的学生宿舍住宿
- 确因特殊原因需校外住宿的，应按照学生校外住宿申请审批程序办理
- 各学院应对学生校外住宿进行**严格控制、严格审批、严格管理**

## 申请条件

学生符合下列条件之一的，可以申请校外住宿：

### 1. 伤病原因

因伤（病）不能正常住校生活的学生，需提供**三级甲等及以上医院的诊断证明**。

### 2. 心理问题

由于身体原因或心理问题且有家长陪读的学生，需提供**三级甲等及以上医院的诊断证明**。

### 3. 已婚学生

已婚学生可申请校外住宿。

### 4. 其他特殊原因

由于其他特殊原因需校外住宿的学生。

## 审批流程

### 申请步骤

1. **下载申请表**
   - 从学生工作部网站下载《山东第一医科大学（山东省医学科学院）学生校外住宿申请审批表》

2. **填写申请**
   - 如实、详细填写表中所列各项内容
   - 经家长签字同意
   - 提供三级甲等及以上医院诊断意见等相关材料

3. **学院审核**
   - 将申请审批表提交所在学院学生工作办公室
   - 辅导员老师应向学生详细了解申请校外住宿的理由、校外房屋的性质和周边环境等情况
   - 向学生说明在校外住宿可能产生的不良后果和个人应承担的责任

4. **学院审批**
   - 经家长签字同意、学生所在学院同意
   - 核实情况并签署意见
   - 在申请表上加盖学院行政章
   - 由学院领导审核批准

5. **签订协议**
   - 由学生所在学院与学生及学生家长签订《山东第一医科大学（山东省医学科学院）学生校外住宿协议书》
   - 报学生工作部备案

6. **办理退宿**
   - 凭退宿通知单办理退宿手续

### 材料要求

- 申请学生应填写《山东第一医科大学（山东省医学科学院）学生校外住宿申请审批表》
- 所填写的情况必须真实
- 申请手续和材料不齐备一律不予批准

### 协议份数

《山东第一医科大学（山东省医学科学院）学生校外住宿申请审批表》和《山东第一医科大学（山东省医学科学院）学生校外住宿协议书》一式三份，学生工作部、学院、学生各留存一份。

## 学生责任及义务

### 法律责任

- 在校外住宿期间，应自觉遵守国家法律法规
- 遵守校纪校规，服从校（院）有关部门、所在学院和实践教学基地组织的各项教学及课外等活动

### 安全责任

- 详细了解校（院）关于学生校外住宿的规章制度及安全提示
- 学生和家长应向学校书面承诺承担校外住宿期间的人身安全和财产损失等全部责任
- **校（院）不承担由于校外住宿而引发的各类纠纷**

### 信息报告义务

- 自觉履行信息报告义务
- 如实向所在学院报告校外住所的详细地址、联系方式
- 定期汇报学习、生活、思想状况

### 作息时间

在校外住宿期间，应自觉遵守校（院）和实践教学基地的作息时间，按时上课，按时参加学校和实践教学基地组织的各项教学及课外等活动。

## 管理规定

### 动态管理

- 各学院负责校外住宿学生的管理
- 要在契约的基础上，强化动态管理
- 加强学生的思想道德教育、法纪教育、安全防范教育
- 切实执行学生近况书面报告制度和辅导员联系制度

### 违规处理

- **学生在申请校外住宿过程中弄虚作假的**，未经校（院）批准私自校外住宿的，一经发现，责令其搬回学生宿舍住宿
- 本人承担由此产生的一切后果
- 校（院）还将视其情节轻重，根据有关规定给予纪律处分

### 期满处理

- 批准校外住宿期满或不再具备申请校外住宿条件的，学生应提前一周向所在学院报告
- 并于期满三个工作日内搬回学生宿舍住宿

### 床位管理

学生校外住宿期间，不能私自回校住宿且寝室床位不再保留。

## 见习实习学生

正常见习实习学生由学院负责统计汇总《山东第一医科大学（山东省医学科学院）学生见、实习期间信息一览表》，报学生工作部备案。

见习实习学生的住宿管理由医院管理与疾病控制部、见习实习单位、学生所在学院管理监督。

## 注意事项

- 因伤病、心理问题等原因申请校外住宿的学生，学生家长原则上应该陪读
- 学生家长应及时了解和关注学生校外住宿情况
- 若遇发生突发事件，校外住宿学生应当立即通知学校并报告
- 未按规定及时返回学校住宿的，自行承担由此产生的损失
