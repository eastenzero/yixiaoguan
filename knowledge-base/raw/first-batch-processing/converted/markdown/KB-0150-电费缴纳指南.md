# 电费缴纳指南

## 缴费方式

学生宿舍电费可通过以下三种方式缴纳：

### 1. 完美校园APP

- 下载并登录完美校园APP
- 进入"电费充值"模块
- 选择宿舍楼栋和房间号
- 输入充值金额并支付

### 2. 支付宝小程序

- 打开支付宝搜索"完美校园"
- 进入小程序后选择"电费充值"
- 按提示完成缴费

### 3. 微信校园码

- 微信搜索"完美校园"小程序
- 选择学校：山东第一医科大学济南校区
- 输入姓名-学号-密码（默认身份证后六位，"X"用"0"代替）
- 进入后选择电费充值功能

## 电控功能说明

- 宿舍电费采用预付费制，余额不足时自动断电
- 充值后实时到账，自动恢复供电
- 可在APP中查询用电记录和余额
- 建议保持电费余额充足，避免突然断电

## 注意事项

- 电费单价以学校最新通知为准
- 如遇充值问题，请联系后勤管理部
- 学生应自觉节约水电，离开房间时应关闭室内所有灯光
- 离开房间时切记关好门窗，锁好房门
