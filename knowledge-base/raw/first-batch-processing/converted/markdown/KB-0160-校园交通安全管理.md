# 校园交通安全管理

## 电动自行车管理

### 车辆标准

校内行驶电动自行车必须符合《电动自行车安全技术规范》（GB17761-2018）国家标准：

- 具有脚踏骑行能力
- 电驱动行驶时，最高设计车速不超过25km/h
- 装配完整的电动自行车质量不超过55公斤
- 蓄电池标称电压不超过48V
- 电动机额定功率不大于400W

**不符合以上任意一条要求的电动自行车，都将被视为"超标车"，禁止在校内使用。**

### 登记注册

- 按照非必要不使用电动自行车的原则，严格控制校园电动自行车数量
- 确需使用的，师生员工电动自行车必须符合国家标准并有正规牌照
- 按照规定流程进行线上登记注册，安全保卫部线上审批通过后须在车辆指定位置悬挂校园通行证
- 校园通行证按照"一人一车一号一证，无证禁用，违规销证"的原则管理

### 行驶规定

- 驾驶电动自行车在校园道路行驶，应当遵守国家道路交通安全法律、法规
- 在校园道路行驶的电动自行车应靠道路右侧**顺向行驶**
- 车速**不得超过25公里/小时**
- **禁止逆向行驶**，禁止在人行道行驶
- **禁止违规载人**
- 骑行时不得有使用手机、骑快车、相互追逐、急转猛拐、双手脱把等有碍交通安全的行为

### 停放规定

- 电动自行车应按划定区域有序停放
- **严禁在消防通道、主干道、人行斑马线、草坪等禁停区域停放**
- 不得影响校园交通秩序和其他车辆停放
- 对于违规停放的电动自行车，安全保卫部有权进行统一清理

### 充电规定

- **严禁将电动自行车（电动平衡车、电动滑板车）蓄电池带入办公楼、宿舍楼、实验楼、教学楼、图书馆、餐厅、商业区等建筑物内存放及充电**
- **禁止私拉电线给电动自行车或蓄电池充电**
- 损坏报废的电动自行车，车主应及时办理销证手续并及时将车辆清退离校

### 记分管理

电动自行车违规处理实行**累积记分管理制度**：

- 以一年（取得校内通行授权）为周期，满分12分
- 一个周期内累积扣除达到6分，将车主信息通报所在单位
- 累积扣除达到12分，注销校园通行证

**扣分标准**：

| 违规行为 | 扣分 |
|----------|------|
| 不按交通标识、标线行驶 | 2分 |
| 不按规定停放 | 2分 |
| 不服从交通管理人员指挥 | 2分 |
| 双手脱把、使用手持电话 | 6分 |
| 曲折竞驶、逆向行驶、超速行驶 | 6分 |
| 私拉插线板、飞线充电 | 6分 |
| 伪造校园通行证 | 6分 |
| 非法改装车辆 | 6分 |
| 将车载蓄电池带入室内充电 | 12分 |
| 因违规停放、充电造成不良影响 | 12分 |

## 机动车管理

### 入校查验

- 校（院）实行入校查验制度，进出校园的机动车须自觉接受门卫查询和管理
- 禁止无关车辆和人员进入校园
- 执行任务的特种车辆（公安、消防、医疗救护、工程抢险等）凭有效证件可以通行

### 行驶规定

- 机动车在校园内须遵守国家交通法规和校（院）相关规定
- 服从校（院）保安人员的指挥、管理
- 机动车时速**不超过20公里/小时**
- 校园内**禁止大排量摩托车、超标电动车通行**
- 校园内**禁止练习驾驶机动车**
- 机动车应当避让行人和非机动车，**禁止鸣笛**

### 停放规定

- 进入校园的机动车和非机动车，按规划路线和交通标志行驶
- 在停车区（位）有序停放
- **严禁在道路两侧、非停车区（位）内停车**

## 自行车管理

- 学生自行车、电动车等要按照指定地点停放
- **禁止乱停乱放或推进公寓**
- 校园内的自行车管理参照电动自行车管理办法执行

## 违规处理

- 对违反交通安全管理规定的，按校（院）调情节轻重处理
- 可给予警告、约谈、通报批评
- 对直接责任人和相关负责人给予批评教育直至记过处分
- 因违规充电引发火灾安全事故的，依法追究刑事责任

## 安全提示

- 驾驶车辆时应注意力集中，不玩手机
- 通过路口时应减速慢行，注意行人
- 雨天路滑应谨慎驾驶
- 夜间行车应开启车灯
- 发现交通安全隐患应及时向安全保卫部报告
