# 勤工助学指南

## 岗位类别

勤工助学岗位按工作时长和性质分为：

### 学期岗

- 工作任务具有延续性，需持续2个月以上的工作岗位
- 每学期初统一申报设置，如无特殊情况，学期内不再调整
- 按月统计工时数发放酬金
- 原则上每月30工时左右，最高不超过40工时

### 短期岗

- 工作任务相对单一，通过一次或几次勤工助学活动即可完成的工作岗位
- 根据实际工作需要，提前3天申请
- 按照实际工时数计算报酬

### 寒暑假岗

- 寒暑假、节假日的勤工助学岗位需提前1-2周单独申报

## 岗位报酬

- 勤工助学岗位按照工作时长统计发放报酬
- 原则上**不低于12元/工时**
- 学期岗按标准全额发放，考核不合格的酌情扣减

## 岗位设置原则

1. **立足校园、服务社会**
2. **学有余力、自愿申请、信息公开、扶困优先、竞争上岗、遵纪守法**
3. 在不影响正常教学秩序和学生正常学习的前提下有组织地开展

## 岗位类型

勤工助学岗位以以下类型为主：

- 教学助理
- 实验室助理
- 科研助理
- 行政管理助理
- 学校公共服务

## 工作时间限制

- 学生参加勤工助学活动的时间原则上**每周不超过8小时**
- **每月不超过40小时**
- 寒暑假勤工助学时间可根据具体情况适当延长

## 申请流程

### 岗位申报（用人单位）

**学期岗**：
1. 每学年开始，用工部门填写《勤工助学固定岗位设置申请表》
2. 明确岗位性质、数量、工作时间、对学生能力的要求
3. 确定岗位的具体工作任务
4. 资助管理中心审核后，报学校学生资助工作领导小组审批

**短期岗**：
1. 用工部门填写《勤工助学临时岗位设置申请表》
2. 提前三天将申请表送至学生资助管理中心

### 学生申请

1. 学生根据学生资助管理中心、学院发布信息填写《学生勤工助学申请表》
2. 说明个人经济、学习和生活情况
3. 经辅导员、学院审查同意并签署意见
4. 持学生证和申请表到用工单位报名面试
5. 用工单位根据岗位要求、学生意愿和条件择优录用

## 录用原则

- 勤工助学实行用工单位与申请学生**双向选择**
- **同等条件下优先考虑家庭经济特殊困难学生**
- 原则上，一个学生不可同时在校内多个勤工助学岗位工作（临时岗位除外）
- 每个岗位原则上只聘用一个学生
- 每个学生原则上每个学期只能申请并获得1个学期岗岗位

## 岗位管理

### 管理原则

- 本着"**谁用人，谁管理，谁负责**"的原则
- 用工单位要对学生做好培训和管理

### 考核要求

- 用工单位应对勤工助学学生的工作进行严格考核
- 考核等级为合格、不合格
- 考核合格的按劳酬标准全额发放
- 考核不合格的酌情扣减
- 每月5日前将上月考核结果及时准确报送学生资助管理中心

### 辞退规定

- 对考核不合格者由用工单位向学生资助管理中心提出辞退建议
- **两次考核不合格者取消其在校期间的勤工助学资格**，停止其勤工助学活动

## 学生职责

1. 认真学习有关文件规定，明确岗位要求
2. 工作中服从岗位管理，认真履行职责
3. 按时保质完成岗位工作
4. 妥善处理勤工助学与学习的关系，确保正常学习不受影响
5. 如需离岗，须经用工部门同意，报学生资助管理中心备案

## 禁止事项

- 各单位、各部门不得组织、安排学生从事过重、有毒、有害等危及学生身心健康的危险性工作
- 涉密工作不宜设立勤工助学岗位
- 不得独立承担单位或部门主体性工作

## 材料报送时间

- 9月中旬前报送《勤工助学岗位申请表》
- 9月中旬前报送《学生勤工助学申请汇总表》
- 10月前在"学工平台"完成学生信息更新
