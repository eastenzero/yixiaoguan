# 学费缴纳指南

## 缴费时间

每月**25-30日**期间可登录山东第一医科大学（山东省医学科学院）校园统一支付平台进行缴费。

## 缴费方式

### 1. 中国农业银行电子银行缴费

- 使用学校配发的农业银行金穗借记卡缴费
- 银行卡随入学通知书邮寄给新生

### 2. 微信支付

- 通过微信在校园统一支付平台完成缴费

## 缴费项目

根据学分制相关收费文件要求，学费由两部分组成：

### 1. 专业注册学费

- 按学年计收
- 每学年初向学生收取

### 2. 学分学费

- 按学期计收
- 实行先选课后缴费
- 在学生完成选课后即时收取

## 不同学生缴费说明

### 全额缴费的学生

需缴纳专业注册学费和住宿费。

### 办理生源地助学贷款的学生

不需缴纳专业注册学费和住宿费。

### 建档立卡学费减免的学生

- 需缴纳住宿费
- 不需缴纳专业注册学费

## 电子票据打印

在山东第一医科大学（山东省医学科学院）校园统一支付平台完成缴费后直接生成缴费电子票据，可自行打印。

## 银行卡使用说明

### 学宿费缴纳银行卡

- 校（院）为新生开立一张**中国农业银行金穗借记卡**
- 用于缴纳学费和住宿费
- 此卡随入学通知书邮寄给新生，请务必妥善保管
- 银行卡初始支付密码为"**111111**"
- 接到卡后请尽快持本人身份证到当地农业银行柜台办理银行卡激活
- 银行卡激活后，请立即修改密码

**修改密码方式**：
1. 本人持有效身份证件到当地农行网点柜台办理
2. 通过农行自助设备(如自动取款机ATM、查询机BCM)修改

**咨询电话**：
- 中国农业银行客服电话：95599
- 农行咨询电话：0538-6021760，0531-82621567

### 资助发放银行卡

校(院)为新生开立一张"山东第一医科大学(山东省医学科学院)·招商银行"专属联名借记卡，用于：
- 发放奖助学金
- 发放助学贷款
- 发放勤工助学、困难补助等学生资助
- 办理保险理赔等

## 学分与学费标准（示例）

以公共事业管理专业为例：

| 项目 | 内容 |
|------|------|
| 学制 | 4年 |
| 最低修读学分 | 165分 |
| 学分学费标准 | 100元/学分 |
| 专业注册学费 | 1375元/学年 |
| 住宿费（泰安校区） | 1300元/学年 |

具体学分学费标准以各专业人才培养方案为准。

## 注意事项

- 请在规定时间内完成缴费，以免影响注册选课
- 办理助学贷款的学生应及时提交贷款证明材料
- 银行卡密码应及时修改，确保资金安全
- 如有缴费问题，请联系计划财务部
- 缴费后请妥善保存电子票据
