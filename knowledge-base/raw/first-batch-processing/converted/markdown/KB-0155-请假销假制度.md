# 学生请假销假制度

## 请假类型

### 1. 病假

- 学生因病不能参加学习活动的，应持校医院或指定医院诊断证明办理请假手续
- 突发疾病可先口头请假，后补交证明

### 2. 事假

- 学生因特殊情况需要请事假的，应说明事由
- 事假原则上应尽量安排在节假日

### 3. 其他请假

- 因公事、参加比赛、学术活动等需要请假的，凭相关证明办理
- 节假日外出、晚归等应按规定请假

## 请假流程

### 短期请假（1-3天）

1. 学生填写请假申请表
2. 辅导员审批
3. 报学院备案

### 中长期请假（3天以上）

1. 学生填写请假申请表，附相关证明材料
2. 辅导员审核
3. 学院分管领导审批
4. 报学生工作部备案

### 特殊情况

- 因伤病需校外住宿的，需提供三级甲等及以上医院的诊断证明，经学院审批、学生工作部备案
- 应征入伍的，按国家相关规定办理保留学籍手续

## 销假规定

### 及时销假

- 学生请假期满后应及时销假
- 逾期不归且未续假者，按旷课处理

### 续假规定

- 请假期满仍不能返校的，应提前办理续假手续
- 续假手续与请假手续相同

## 考勤管理

### 考勤要求

- 学生应当按时参加教育教学计划规定的活动
- 不能按时参加的，应当事先请假并获得批准
- 无故缺席的，根据学校有关规定给予批评教育，情节严重的，给予相应的纪律处分

### 考勤计算

- 凡缺课累计达到该课程教学时数三分之一者，取消其期末考试和补考资格，成绩按零分计
- 未经批准连续两周未参加学校规定的教学活动的，学校可予退学处理

## 公寓晚归请假

- 学生要严格遵守公寓作息时间制度
- 不归者须按规定向所在学院请假
- 公寓来访人员须经公寓管理人员检查询问并登记后方可进入公寓

## 违规处理

- 未经请假擅自离校者，按旷课处理
- 请假逾期不归者，视情节给予批评教育或纪律处分
- 未经批准私自校外住宿的，责令其搬回学生宿舍住宿，并视情节轻重给予纪律处分

## 注意事项

- 请假应提前办理，紧急情况可先口头请假后补手续
- 请假材料应真实有效，弄虚作假的按违纪处理
- 请假期间应注意安全，保持通讯畅通
- 外出期间如发生意外，应及时向学校报告
