# 2025“外研社•国才杯”“理解当代中国”全国大学生外语能力大赛

# （英语组）山东第一医科大学校赛通知

**各教学机构：**

2025“外研社·国才杯”“理解当代中国”全国大学生外语能力大赛以习近平新时代中国特色社会主义思想为指导，深入贯彻党的二十大精神，旨在推动“三进”工作、培育外语人才，讲好中国故事、服务国际传播、促进文明交流互鉴、为构建人类命运共同体积极贡献力量。

本届大赛英语组校赛设有综合能力、演讲、笔译和口译等分赛项。大赛以“理解中国，沟通世界”为主题，引导选手深入领会习近平新时代中国特色社会主义思想的核心要义，理解中国之路、中国之治、中国之理，涵养家国情怀、全球视野，拓展知识广度、思想深度，展现外语能力、跨文化能力、思辨能力、创新能力等综合素养。

根据大赛组委会的要求，现在全校范围内组织“2025年‘外研社·国才杯’（英语组）校内初赛”，初赛设综合能力、演讲、笔译和口译四个分赛项。**综合能力、口译和笔译赛项均为线下比赛，各赛项分别设有泰安校区和济南校区两个考场；演讲赛项的英语专业组和大学英语组分别组织初赛、复数和决赛。**具体各赛项参赛方式和要求详见附件，请有意向参赛的同学仔细查看，如有问题，请联系对应赛项的负责老师。

教务部 外国语学院

2025年9月15日
