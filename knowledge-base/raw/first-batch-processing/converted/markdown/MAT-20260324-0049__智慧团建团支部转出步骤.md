1.  <span class="mark">升学</span>：填写自己研究生学校对应专业班级的团支部

2.  <span class="mark">工作</span>：

    工作单位已建立团组织：如果工作单位已经建立了团组织，那么团员应将团组织关系转至工作单位团组织。

    工作单位未建立团组织：若工作单位尚未建立团组织，团员应将团组织关系转至其经常居住地或工作单位所在地的社区、村团组织。

3.  <span class="mark">搬迁</span>：例如，团员毕业后未落实工作单位，且家庭住址发生了变化，从一个城市搬到了另一个城市，此时可以选择“搬迁”作为转出原因，将团组织关系转接到新家庭住址所在地的社区、村团组织或乡镇街道团组织。另外，如果团员在工作期间因家庭搬迁，导致经常居住地发生改变，也可以以“搬迁”为由，将团组织关系转接到新的经常居住地团组织。

4.  <span class="mark">转入特殊单位</span>：主要包括参军入伍和到涉密单位工作两种情况。

    当团员参军入伍或到涉密单位工作时，原就读学校团组织或团员本人可在智慧团建系统上发起组织关系转接申请，省级团委审核通过后，团员将进入特殊单位专属库集中进行管理。转往特殊单位团组织时，无需选择转入组织，业务提交后由省级团委管理员负责审批 。
