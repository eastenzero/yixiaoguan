山东省学生资助一网通办系统

学院、班级操作说明

# 

# 一、学院、班级系统登录

使用<span class="mark">谷歌浏览器或360浏览器</span>（推荐使用极速模式），直接访问一网通办平台，学院、班级账号由学校下发，**首次登录需修改密码，请妥善保管密码。**学院、班级账号密码可以分别通过学院、学校端进行重置。

网址：https://sdxszz.sdei.edu.cn/srplatform-lxgl/

# 二、班级评议

登录班级账号，点击顶部【困难认定】再点击左侧【评议小组评议】，在【申报树】选择要评议的学生，右侧页面显示该学生对应的申请书及量化指标，点击顶部的标签页可切换预览对应材料。若学生填写的量化指标有错误，可直接对错误项进行修改**<span class="mark">（学生提交后，只有班级评议可修改指标项，院系和学校仅可修改档次，无法修改指标），</span>**修改完成且确认无误后，点击最下方的【保存评议结果】按钮。

###### 特殊困难类型学生属于免申请范围，不出现在班级账户【申报树】中，不需进行评议步骤。系统默认特殊困难类型学生为“特别困难”等级，如果学生家庭经济好转，认定为“困难”等级，可以由学院账户在助学金评审环节调整。

<img src="media/image2.png" style="width:5.48958in;height:2.92569in" alt="QQ_1757654609475" />

当本班级所有学生均评议完成后，点击左侧【班级评议提交】按钮，右侧页面点击【上传评议报告】按钮下载模板，填写完成后上传系统。确认无误后点击【提交】按钮，**提交后数据将不可更改。**

**<span class="mark">注意：请确保本班级所有申请学生提交后，再提交班级评议。班级评议提交后，将无法评议新提交的申请。确需修改的，由学院账户退回处理，退回班级报告可能会影响二级学院认定调整的档次，请谨慎操作。</span>**

<img src="media/image3.png" style="width:5.67292in;height:2.20903in" alt="QQ_1753685205460" />

# 三、学院认定

登录学院账号，点击顶部【困难认定】再点击左侧【认定小组认定】，右侧页面显示所有班级评议提交的数据。点击身份证号，可查看该学生填写的申请书及上传的佐证材料。学院可查看学生的自评打分及班级评议小组打分，若同意班级评议结果，则点击【同意评议结果】按钮。**若需要修改推荐档次，则选择该条数据，点击选择对应的档次，再点击【认定学生】保存修改后档次。**确认无误后，点击【上传评议报告】按钮下载模板，填写完成后上传系统。确认无误后点击【提交】按钮，提交后数据将不可更改。

**注意：请确保本院系所有班级提交后，再提交院系认定，院系提交后，将无法认定新提交的评议数据。**
