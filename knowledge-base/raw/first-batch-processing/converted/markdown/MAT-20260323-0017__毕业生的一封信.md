**亲爱的同学们：**

**见字如晤，展信舒颜！六月渭湖，波光粼粼，芳名湖畔，骊歌轻起。又是一年毕业季，在此，全校师生向即将毕业、离开母校的你们致以最热烈、最诚挚的祝贺！祝贺你们即将开启人生新的旅程，奔赴下一片山海！**

**时光知味，岁月沉香。大学是青春中最美好的时光，但和疫情狭路相逢的你们，却被迫与封控和核酸撞了一个满怀。四年时光，虽有遗憾，但依然留下了诸多弥足珍贵的回忆：图书馆里你们徜徉书海的身影，运动场上你们奋勇争先的英姿，课堂上、宿舍里你们灿烂的笑容依然历历在目；你们也一定记得渭湖中肆意游曳的天鹅，杨梅山上花团锦簇的桃李园，壶兰餐厅的美味佳肴，芳名湖畔的四季美景……四年来，你们更是同母校并肩见证了百年大党的光辉历程与初心如磐，携手亲历了紫霄校区中、西区先后投入使用，共同迎来学校获批教育部硕士学位授予单位，共襄“百年办学 廿年本科”校庆盛典……**

**如今，夏花绚烂，清风徐来，而你们也即将离开莆院，背上行囊，去寻找诗与远方。临别之际，母校有几点希望寄予你们：**

**希望你们崇德修身、感恩奋进、怀赤子心。人无德不立，立德为先，修身为本。面向未来，希望你们涵养个人品性、锤炼过硬人格、引领青春风尚，做到“明大德、守公德、严私德”，自觉树立和践行社会主义核心价值观，扣好迈向社会的第一粒扣子。面向未来，希望你们常怀感恩之心和爱校荣校之情，弘扬“厚德、博学、慎思、笃行”的校训精神，一声念师长，一声念同窗，一朝话离别，互道珍重赴新期，各自奋斗，努力成为一个有益于国家、有益于社会的人。**

**希望你们胸怀理想、志存高远、立栋梁志。心有所信，方能行远。面向未来，希望你们从脚下这片沃土出发，树立为祖国、为人民赤诚奉献的坚定理想，将国之所需、民之所向作为人生的奋斗航向，将自己的小我融入国家之大我，与国家同呼吸、与人民共命运、与时代齐奋进，用昂扬向上的行动彰显爱国情、强国志、报国行，用青春之我、奋斗之我印刻出当代中国青年应有之样，努力成为堪当民族复兴重任的时代新人。**

**希望你们勤学苦练、立行有为、蓄报国能。面向未来，希望你们取法于上，终身学习，勤学善思、知行合一，不断扩充新知，不断奋力上进，不断扩大视野，练就过硬本领。希望你们积极投身经济建设主战场、改革创新最前沿，脚踏实地，艰苦奋斗，敬业奉献，争先创优，在实践中经风雨、见世面、壮筋骨、长才干，努力成为行业骨干、青年先锋，用勤劳的双手、一流的业绩书写人生华章。**

**同学们，山水一程，有幸遇见。你们即将满载一船星辉，奔赴星辰大海。愿你们有前程可奔赴，亦有岁月可回首；“此心安处是吾乡”，学校永远是你们的家；愿你们历尽千帆，归来仍是少年。**

**衷心地祝愿同学们：青云万里，扶摇直上，一生幸福！**
