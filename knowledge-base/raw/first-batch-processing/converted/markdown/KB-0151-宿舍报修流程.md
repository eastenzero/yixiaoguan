# 宿舍报修流程

## 报修范围

学生公寓内的以下设施维修由校（院）后勤管理部负责：

- 电路、灯具、插座
- 风扇、淋浴器
- 洗衣机、直饮水机
- 热水器、空调等设备

## 报修流程

### 1. 提交报修申请

- 学生发现宿舍设施故障时，应及时向后勤管理部报修
- 可通过学校指定的报修渠道（如后勤服务APP、电话等）提交申请
- 详细描述故障情况和所在宿舍位置

### 2. 维修安排

- 后勤管理部接到报修后，将安排维修人员上门检修
- 紧急故障（如水管爆裂、电路故障等）将优先处理

### 3. 维修完成确认

- 维修完成后，学生应检查设施是否正常使用
- 如有问题应及时反馈

## 重要规定

### 学生不得擅自处理

学生公寓内的电路、灯具、插座、风扇、淋浴器、洗衣机、直饮水机、热水器、空调等设备的维修及故障处理由校（院）负责，**学生个人不得擅自处理**。

### 损坏赔偿

- 学生不得损坏校（院）统一配备的公共设施设备
- 因学生个人原因造成设施损坏的，由当事人负责赔偿
- 损坏赔偿标准以学校最新规定为准

## 注意事项

- 报修时应提供准确的宿舍地址和联系方式
- 维修期间请配合维修人员工作
- 发现设施故障应及时报修，避免造成更大损失
- 私自拆修造成的人身伤害或财产损失，由学生自行承担
