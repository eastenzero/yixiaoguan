# 水电管理与节约要求

## 水卡与淋浴系统

### 无单独水卡

学校**无单独水卡**，校园卡集成淋浴、直饮水等功能。学生可使用实体校园卡或虚拟校园码进行淋浴和直饮水消费。

### 淋浴使用

- 使用校园卡在宿舍区淋浴设备上刷卡使用
- 按实际用水量计费
- 使用完毕后确保水阀关闭

### 直饮水

- 学生公寓按照上级文件规定，实行定人定量饮水
- 使用校园卡在直饮水机刷卡取水

## 节约水电要求

### 学校规定

住宿学生要**自觉节约水电**，具体要求如下：

1. **离开房间时关闭所有电源**
   - 关闭室内所有灯光
   - 关闭风扇、空调等电器
   - 关闭插座电源

2. **节约用水**
   - 洗漱时避免长流水
   - 发现水龙头漏水及时报修
   - 合理使用淋浴，控制洗浴时间

3. **安全用电**
   - 严禁使用大功率电器（热得快、电饭煲等）
   - 严禁私拉乱接电线
   - 严禁在宿舍内为电动车充电
   - 严禁使用明火

## 违规处理

### 用电违规

- 宿舍内私拉乱接电线、使用明火、存放或使用违章电器，未酿成火情或者火灾者，给予警告处分
- 因以上行为引起火灾造成严重后果者，除赔偿经济损失外，给予记过及以上处分

### 损坏赔偿

学生损坏水电设施的，应按价赔偿；情节严重的，按学校相关规定处理。

## 供应保障

后勤管理部负责保证学生公寓水、电、暖的正常供应，做好公寓设施设备的维护、维修和专项工程的实施。

## 注意事项

- 宿舍电费不足时应及时充值，避免断电
- 发现水电设施故障应及时报修
- 学生应树立节约意识，养成节约水电的良好习惯
- 水电单价以学校最新通知为准
