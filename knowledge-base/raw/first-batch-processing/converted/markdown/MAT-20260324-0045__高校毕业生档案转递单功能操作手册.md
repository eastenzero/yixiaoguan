山东省2023届高校**非**师范类毕业生

[“档案转递单](http://www.mohrss.gov.cn/xxgk2020/fdzdgknr/zcfg/gfxwj/jy/202306/P020230616460172310570.doc)”下载操作流程

高校通过“全国高校毕业生毕业去向登记系统”（dj.ncss.cn），为已完成确认毕业去向信息的毕业生，生成“档案转递编号”。生成操作完成后可批量下载“档案信息传递列表”。高校登录“山东高校毕业生就业信息网”，导入“档案信息转递列表”，系统自动生成“档案转递单”，由学校直接下载或打印。具体操作如下：

1.  生成“档案转递编号”

    高校登录“全国高校毕业生毕业去向登记系统”后，点击“户口信息核验列表”菜单下的“档案信息转递列表”如图（1）所示。使用“批量生成转递编号”按钮可以批量生成已确认毕业生的“转递编号”或者在列表中使用“生成转递编号”功能生成单位学生的“转递编号”。

<img src="media/image1.jpeg" style="width:6.13125in;height:7.71597in" alt="57a95bf9aa1467d1323786c6209751e" />

图（1）

2.  “档案信息转递列表”下载

    高校在“档案信息转递列表”图（1）所示中，查询条件选择“已生成”档案编号选项，查询后使用“批量下载”功能下载档案信息转递列表如图（2）所示。

<img src="media/image2.jpeg" style="width:6.13125in;height:4.075in" alt="60ceff7401f5a4f845f6b1044437aee" />

图（2）

3.  导入“档案信息转递列表”

    高校登录“山东高校毕业生就业信息网”，在“档案寄递管理”模块下，点击“档案信息转递列表”菜单，显示毕业生档案信息转递列表如图（3），使用“导入档案信息转递列表”功能，进行导入。信息导入后，再使用“批量下载档案转寄单”功能，系统会批量生成档案转递单完毕后自动压缩打包下载。

<img src="media/image3.png" style="width:6.14028in;height:1.77639in" alt="QQ截图20230628202659" />

图（3）

4.  “档案转递单”下载和打印

“山东高校毕业生就业信息网”根据档案信息自动生成“档案转递单”，一人一单。使用“批量下载档案转寄单”功能可批量下载“档案转递单”，如图（4）所示。
