“家校协力，共育未来”

医药管理学院2025级新生家长见面会议程

**一、活动目的**

白驹过隙，又一批大一新生开始甜蜜而温馨的大学生活，为了协助新生家长更快更好的了解我校我院的各项方面，表现医药管理学院的团结与诚心，体现管理学院的风采，增强家校互联，帮助家长们对学院的目标规划、学生们的未来发展有所了解，促进家长与学生、学校之间的亲密度，使家长们更好的融入学生的全新阶段，与学生共同进步、一起成长。

**二、活动主题**

“家校协力，共育未来”

**三、活动时间 ：**2025年8月29日下午2点半

**四、活动地点：**医药管理学院大会议室A2015（南楼南门进）

**五、参会人员：**学院苏书记、赵书记、团委副书记杨苗苗、新生辅导员安静。赵书记主持会议。

**六、活动内容：**

1、赵书记开场，介绍会议背景、目的，介绍到场嘉宾，明确会议流程；

2.  学院党委书记苏宝利致欢迎辞、介绍校院的基本情况；

4、互动答疑，家长针对学院教学、管理、学生生活等问题提问，书记、院长及工作人员现场解答，确保家长疑问得到回应；

5、会议总结，赵书记梳理会议核心内容，重申学院与家长协同育人的重要性，告知后续家校沟通渠道（如联系新生辅导员安静老师、辅导猫绑定家校互联联系方式）
