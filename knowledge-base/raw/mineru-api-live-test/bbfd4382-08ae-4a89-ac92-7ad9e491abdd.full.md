# 2024 届普通本科毕业生毕业工作安排方案

为扎实做好2024届普通本科毕业生工作，保证毕业生安全、文明、有序毕业离校，依据校（院）实际情况，特制订本方案。

# 一、基本数据

校（院）2024届普通本科毕业生共计 5162人。其中，济南校区 2238 人，泰安校区 2924 人。

# 二、组织保障

校（院）主要领导对毕业工作总负责，分管领导具体负责。

党委办公室（校院长办公室）、组织部、宣传部、新闻中心、学生工作部、教务部、医院管理与疾病控制部、研究生部、计划财务部、安全保卫部、后勤管理部、团委、泰安长城路校区管委会、现代教育技术中心、图书馆、毕业生所在学院等部门、单位根据工作职责及任务分工，各负其责、相互配合，具体承担相应毕业工作。

# 三、工作原则

（一）坚持安全第一，科学有序组织。  
（二）坚持数字引领，优化流程手续。  
（三）坚持严格管理，热情优质服务。
