## 防幻觉机制说明

根据 `services/ai-service/app/core/llm_chat.py` 实现，防幻觉机制包含以下核心逻辑：

### 1. 检索门控 (_assess_grounding)
- **source_count < 2**: 拒答 (insufficient_sources)
- **best_score < 0.611**: 拒答 (best_score_too_low)  
- **avg_score < 0.58**: 拒答 (avg_score_too_low)
- **无检索结果**: 拒答 (no_sources)

### 2. 拒答文案
```
很抱歉，医小管目前尚未学习到相关说明，请咨询您的辅导员或相关负责老师。
```

### 3. 配置阈值 (config.py)
- `rag_min_score`: 0.55
- `rag_min_best_score`: 0.611
- `rag_min_avg_score`: 0.58
- `rag_min_source_count`: 2
