# 便宜指挥官专用验证 SOP（2026-04-03）

> **定位**：本文件是指挥官 AI 在子 agent 完成任务后，执行 L1 机械核查的标准流程。  
> **边界**：指挥官只做结构/计数/字段名核查（L1），不做语义一致性判断——那是 Cascade（Claude）的职责（L2）。  
> **触发条件**：子 agent 回复 STEP-CHECK 或"阶段任务完成"后，指挥官必须先过一遍本 SOP，再决定是否上报 Cascade。

---

## 核心分工

| 层级 | 执行者 | 负责内容 | 不负责 |
|------|--------|---------|--------|
| **子 agent** | 便宜 AI | 产物生成，STEP-CHECK 自报 | 跨文件口径，风险评估 |
| **L1（指挥官）** | 便宜 AI | 结构核查、计数核查、字段名核查 | 语义一致性，执行风险 |
| **L2（Cascade）** | Claude | 跨文件一致性、风险签字、执行放行 | 机械性生成任务 |

**黄金规则**：L1 只看"数得出来的事"，L2 才看"需要理解的事"。

---

## L1 核查清单（每次必走）

### 第一步：文件是否齐

```
□ 任务要求的每个交付文件是否都已创建？
□ 文件路径是否与"允许修改文件列表"完全一致？（路径错了等于不存在）
□ 有没有额外创建了"允许修改文件列表"以外的文件？（超范围 → 立即上报 Cascade）
```

### 第二步：关键数字是否一致

```
□ 打开每个产物文件，找出所有出现"数量/个数/合计"的地方。
□ 同一个实体（如"A1缺失项"）在不同文件/不同章节中的计数是否完全相同？
□ 把所有数字列成一张表，肉眼比对有无不一致。
```

> ⚠️ 举例（C-003 实际案例）：  
> - 报告正文写"A1 缺失项（8个）"、统计摘要写"需 rebuild_from_source | 7"——这是 L1 就该抓到的矛盾。

### 第三步：必填字段名是否正确

```
□ 子 agent 产物中引用到的字段名，是否与任务说明书中的字段名逐字一致？
□ 重点检查容易笔误的同义字段（如 kb_draft_id / target_kb_id / kb_entry_id）。
□ 如有字段名与任务说明书不符 → 标记为 L1_FAIL，等待 Cascade 裁定。
```

### 第四步：禁止事项是否被违反

```
□ 查看子 agent 的 STEP-EXECUTED 中"实际修改文件"列表。
□ 每一项与"禁止修改文件"对照——有无越界改动？
□ 若有禁止文件被改 → 立即上报 Cascade，不得自行决定是否回滚。
```

---

## L1 核查结果上报模板

指挥官完成 L1 核查后，把以下模板填好，附在子 agent 产物后面，一并提交给 Cascade：

```
【指挥官 L1 核查报告】
核查时间：YYYY-MM-DD HH:mm
被核查任务：[任务ID / 任务名称]

L1-1 文件齐备：✅ / ❌（缺：xxx）
L1-2 数字一致性：✅ / ❌（不一致点：xxx 在 A 文件是 N，在 B 文件是 M）
L1-3 字段名核查：✅ / ❌（问题字段：xxx 实际写成了 yyy）
L1-4 禁止项核查：✅ / ❌（越界文件：xxx）

L1 总体判定：PASS / FAIL
→ PASS：请 Cascade 做 L2 验收
→ FAIL：已挂起，等待 Cascade 裁定
```

---

## 什么情况必须上报 Cascade（不要自己决定）

| 触发条件 | 原因 |
|---------|------|
| 任何 L1 项标记 ❌ | 需要 L2 语义判断才能决定是否重做 |
| 子 agent 写了 BLOCKED | 信息不足，需要更高权限决策 |
| 数字"说得通但感觉不对" | 语义一致性是 L2 的事 |
| 任务涉及高风险动作（重命名/批量字段更新） | 必须 Cascade 签字后才能放行 |
| 子 agent 要求"继续优化"或"顺手修了别的" | C-002 anti-pattern，立即叫停 |

---

## 什么情况指挥官可以自行放行（不用打扰 Cascade）

仅限以下情况**全部满足**时：
1. L1 四项全 ✅
2. 任务类型是**文档生成/初稿生成**（非重命名/批量写库）
3. 交付物与上一版相比只新增内容、未删除已有内容

---

## 给指挥官 AI 的一句话提示

> 你是"结构核查员"，不是"内容审阅员"。你只需要数数字、核字段名、查文件是否齐——判断内容对不对、风险高不高是 Cascade 的工作。遇到任何拿不准的，直接上报，不要猜，不要自行判断通过。
