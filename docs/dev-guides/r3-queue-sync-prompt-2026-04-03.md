- **7 个 MAT 条目**（MAT-0024~0029, MAT-0055）：`kb_draft_id` 指向已被 A3 占用的旧 ID（0061~0067），草稿文件不存在，需分配新 A1 ID 并重建
- **5 个 candidate-pool 条目**（无 material_id）：`kb_draft_id` 指向 0068~0072，原始素材归属需人工确认

以上留给 R4-A1-REBUILD 任务处理。
