## 【当前问题——你要解决的】

当前学生端（`apps/student-app/src/pages/`）的 UI 存在以下严重视觉问题：

1. **页面背景**：纯白色，应为参考图中的 `#f0f7f5`（极浅冷绿，几乎白色但有质感）
2. **首页快捷入口图标**：使用了随机彩色 emoji 风格图标（AI机器人、学校、剪贴板），应全部换为参考图风格的**线条单色图标**（主题色绿色），且背景为浅色圆形
3. **tabBar 图标**：全部显示为破损占位图，需要替换为可显示的 SVG 内联图标或 Unicode 字符图标
4. **卡片样式**：当前阴影偏重、边框明显，参考图中卡片用细微阴影（`0 2px 12px rgba(0,0,0,0.06)`），圆角 `24rpx`，无明显边框
5. **字体层级**：当前标题/正文/副标题的字号、颜色层次不清晰
6. **首页欢迎区**：背景色OK但内容排版需要优化，参考图中欢迎语是大号字、加粗，副标题是小号灰色
7. **AI 对话页气泡**：ai-service 的气泡和学生气泡样式需要按参考图优化
8. **表单输入框**：当前是灰色背景框，参考图是"Soft Fields"——无边框，仅有浅色背景，聚焦时有淡绿色光晕
