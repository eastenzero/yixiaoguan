## 【核心目标与考核点】

这次重构的重点在于 **绝佳的视觉质感**、**行云流水般的动效** 以及 **完美的功能交互**。

### 1. 核心骨架：AI 咨询页 (Chat) —— 倾注全力的核心
- **流式输出 (SSE) 完美实现**：必须使用原生的 `fetch` + `ReadableStream` 配合 `TextDecoder` 来解析流式数据（`POST /api/chat/stream`）。要实现真正的打字机逐字输出效果，并妥善处理 `{"chunk": "...", "is_end": false}` 这样的数据帧。
- **Markdown 极致渲染**：请引入 `markdown-it`，对 AI 的回复进行卓越的 Markdown 呈现。区分出标题、列表、加粗、代码块的优雅样式，让它看起来像最优秀的 AI 对话产品。
- **设计细节与动效**：
  - 气泡需要有恰到好处的圆角和细微的阴影。
  - 进场动画：AI 的回答和学生的提问，弹出时必须有丝滑的淡入上浮动画 (`translateY` + `opacity`)。
  - 缓冲思考动画（Thinking）：拒绝简单的三个点，必须是流畅悦目的波浪跳动、脉冲缩放或是更高级的 CSS 过渡。
  - 交互增强：输入框高度必须能在多行文字时自适应；发送按钮需要有输入时的主题色发光态。

### 2. 门面担当：高级感首页 (Home) —— 放飞自我
- 参考图 `_3/screen.png` 和 `_8/screen.png`。原有的约束已被打破，你需要完全按照高级感的设计去重写整个首页。
- **顶级文字排版**：加入根据时间动态判断的“早上好/下午好/晚上好”，姓名字体要巨大、粗体且极具现代感，副标题清晰呈现院系年级（若无则兜底处理）。
- **AI 探索入口 (Hero Section)**：设计一个极其显眼的"问问 AI 辅导员..."大输入框卡片区，配合推荐的快捷话题 Tag。点击后携带内容平滑跨页传参并自动发送给 AI。
- **动态组件流转**：重新设计快捷服务区与通知区，图标必须精美绝伦（可以用优质的数据 URI SVG 或者精心搭配的微拟物组合），不要敷衍。
- **出场动画**：整个首页加载时，各区块应当以级联 (Staggered) 方式优雅地淡入，给人一气呵成的尊贵感。

### 3. 用户自留地：个人中心 (Profile) —— 从零惊艳
- 彻底废弃旧有的呆板个人页，严丝合缝地参考图 `_7/screen.png`。
- **头部身份区**：超大精美圆角的头像或者大排版首字母，加入“已认证学生”等细节 badge，色调采用项目主题绿系的极浅淡色以衬托。
- **功能流转矩阵**：使用双列宫格展示问答历史与申请进度。点击必须配有高级按压回弹动效 (`transform: scale(0.96)`)。
- **设置与通知列表**：精美的列表卡片，图标用主题色浅背景圆形包裹，右侧细小的进入箭头。统一步长与垂直高度，不要出现任何对齐瑕疵。
- **底部信仰留白**："博学而笃志，切问而近思。" 的名言必须呈现出高级编辑语录排版的质感。

### 4. 极致微交互 (Animations) —— 你的拿手好戏
- 你最擅长动效，请在每一个可能发生交互的地方大显身手！
- 给所有按钮、卡片、列表项添加触控反馈（`active` 态的尺寸缩小或透明度反馈）。
- 运用高阶的 CSS 变量驱动和 transition 技巧，使每个页面的触动都富有生命力。
